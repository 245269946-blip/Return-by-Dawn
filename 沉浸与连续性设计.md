# 《逾期之书》沉浸感与连续性设计思考（v0.9.1 纠偏）

> 背景：上一版 demo（夜 A）跑通了"收通知→去图书馆→自由探索→还书闸门→知被遗忘→三选一"的回路，但用户试玩后判断**沉浸感与难度不足，根因是连续性**。本文件先把"怎么实现连续 + 怎么激发探索欲"想清楚，再定第一幕落点。剧本内容暂不展开。

---

## 一、上一版 demo 为什么"不够沉浸"

不是谜题太简单，是**谜题和"我想知道"之间断了钩**。具体四点是：

1. **谜题是"过闸"不是"发现"**：4 个子谜题（拆字/排年份/输日期/拼信）各自是封闭逻辑题，解完只是"可以前进"，玩家没有"我非要知道不可"的拉力。
2. **图书馆是静态布景，不累积**：每一夜像重置——同样的 4 个场景壳子，换套谜题。没有"我回来了"的体感，重复变成了重复劳动，而不是记忆锚点。
3. **管理员是薄 NPC，不是关系**：有且只有开场一句、结尾一句，中间是工具人。穿越到夜 C 的"熟悉陌生人"反转就缺了情感地基。
4. **没有可携带的好奇心**：玩家是为推进而解，不是为知晓而解。Rust Lake 式解密爽点在于"我在窥探一个真实存在过的人的痕迹"，demo 没把这个钩子立起来。

---

## 二、连续性怎么实现（这是纠偏的核心）

### 2.1 把图书馆做成"会记得你的地方"
- **固定 4 个空间，永远在同一栋楼里**：建议固定为 `借阅台 / 阅览区 / 书库深处 / 归还箱（储藏室）`。每夜只换"今晚这里藏了什么"，不换"我在哪"。
- **跨夜累积的可视状态**：一面"已还书目墙"或一本翻开的台账，随夜数增长；某个角落永远亮着的那盏灯位置不变。玩家第二夜进门就看见"我昨晚还的那本在那儿"——**这瞬间的熟悉感就是连续性本身**。
- **管理员引用前夜**："你昨晚也来过""这盏灯你上次说太暗"。人物不重置，关系在长。

### 2.2 空间语法先学会，变化只发生在内容层
- 夜 A 教会玩家地图与操作；之后每夜的快感来自"今晚这片区域藏了什么线索"，而不是"我要去哪点"。
- 感官/谜题形态可以每夜换（声/味/温/影/纸/植物），但**发生地永远在这四个区里**——既保鲜，又不破坏连续性。

### 2.3 管理员是贯穿线，不是道具
- 固定声线、固定小动作、固定收尾"灯还亮着"（前期轻、后期重）。
- 关系微推进：从"陌生人"到"知道你名字"到"会问你一句私人的话"。这条情感连续性能扛住夜 C 的身份翻转。

## 三、探索欲望怎么激发（沉浸的来源）

### 3.1 每夜只立一个中心谜，不立四个散谜
- 每夜的框架不是"解 4 道题"，而是 **"这是谁的书？TA 为什么逾期？"**
- 4 个子谜是**同一条人物故事的四条线索**，拼起来才露出一个人。玩家解谜是为了"知道 TA 的事"，不是"过闸"。

### 3.2 环境叙事（Rust Lake 式）：线索是"可把玩的真物件"
- 线索藏在空间里可被操作的东西中：书页边的批注、夹着的花、杯底的咖啡渍、当书签的票根。点击/拖动/组合才显形——是"窥探"，不是"答题"。
- 每条线索给主角故事的一行，攒到"知被遗忘"那刻拼成完整一人 → 好奇心闭环。

### 3.3 软选择 / 软后果，制造"我想试试另一种"
- 探索中放可选动作（抽屉要打开吗？那页私人日记要看吗？），**不卡进度**，但改变管理员台词 / 你多知道一点 / 少知道一点。
- 这点轻量 replay 欲，也让"自由探索"名副其实。

### 3.4 "记忆闪回"作为固定节拍（你说的"穿插记忆再出去"）
- 某些线索会触发一段**玩家的记忆闪回**——这是"出去"的方式，但用**统一的转场（视觉+音效 cue）**包成可识别的节拍，不是真换地点。
- 闪回永远关于玩家自己的过去，前期当"别人的事"读，夜 C/终章回照成"其实是你"。这既是探索奖励，也是伏笔播种机。

### 3.5 15 分钟单夜节奏（按你给的时长倒推）
| 时段 | 内容 | 目的 |
|---|---|---|
| 0–2 min | 收通知 + 进门（安静、熟悉） | 进入回路，舒适感 |
| 2–11 min | 自由探索，3–4 条线索谜题，难度阶梯上升 |  immersión 主菜 |
| 11–13 min | 收束：拼线索 → 知被遗忘（TA 的真相） | 好奇心闭环 |
| 13–15 min | 三选一 + "灯还亮着"收尾 | 记忆锚点落定 |

## 四、难度与"好玩"从哪来

- **要"aha"不要"grunt"**：偏好观察/空间/物件类谜题（解码一个真实存在的东西），而非纯算术/纯文字。解码成功时像"我破译了一段真实痕迹"。
- **难度阶梯 + 合成闸门**：前 3 个轻量热身，最后的"归还箱"闸门要求**综合本夜线索**才能开——把"探索所得"变成"推进钥匙"，闭合回路。
- **感官每夜换，地点不动**：声/味/温/影/纸/植物轮替，但全在图书馆四区内。

## 五、第一幕修订方案（图书馆内，3–4 夜 → 第二幕）

把你之前看到的"10 个方向"全部**拉回图书馆内**，按"回路固定、内容换皮、伏笔暗埋"重排。第一幕 = 纯 decoy 沉浸期，建立回路与管理员记忆锚点；末尾一夜线索收束，转入第二幕自我探索。

| 夜 | 书名 / 形态 | 馆内发生（同一四区） | 记忆锚点 / 伏笔 |
|---|---|---|---|
| **夜 A**《夹在书里的信》 | 信纸/笔迹谜题（深化现有 demo） | 借阅台收信→书库找夹层→归还箱日期闸门 | 建立回路+管理员；**笔迹首次像你**（当巧合） |
| **夜 B**《满员的书架》 | 唯一空位谜题（按离馆年份疏密推断为谁留的） | 阅览区四排书架找出被故意空出的格位 | 空位借书卡笔迹 = A 同款；**探索中触发一次"出去"的记忆闪回**（统一转场） |
| **夜 C**《雨水收集器》 | 四罐水位+天气记录，拼"最难过的一天"（机关感最强） | 书库深处摆满接雨容器，按刻度推断 | 罐底信纸同源 A；管理员第一次问你"你也在攒什么吗？" |
| **夜 D（桥）**《管理员借走的书》 | 四处在馆内找管理员私物，日志拼"TA 是谁" | 全馆搜索管理员的物件/日志 | 日志末页写"这本书是你借的"；**归还箱闸门显示借阅人=你** → 第二幕 |

- 夜 A–C 是"纯别人书"的好玩期，重复回路、累积熟悉感；
- 夜 D 是桥：线索第一次明显收束，管理员行为开始"不对劲"，拍子照旧但某一拍**质感第一次塌**——自然滑入第二幕自我探索。

> 关于"穿插记忆再出去"：仅作为夜 B/C 探索中的**记忆闪回节拍**出现（统一转场、不换地点），既是探索奖励也是伏笔播种，不破坏图书馆设定。

## 六、下一步需要你拍板的点

1. **图书馆四区固定为 `借阅台/阅览区/书库深处/归还箱`** 可以吗？还是你想用别的区名/数量？
2. **第一幕就按 夜 A / B / C / D（4 夜）** 排，夜 D 做桥接——可以吗？还是想先只做 3 夜（A/B/C）更紧凑？
3. 确认后，我先把**夜 A 按上面"中心谜+环境叙事+软选择+15min 节奏"重写内容模块**（demo 引擎已支持喂不同内容），你试玩定调，再依次做 B/C/D。
