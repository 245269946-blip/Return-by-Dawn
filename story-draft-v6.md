# 《逾期之书》故事设计 v6（必然告别型 BE · 纯净定稿）

> 本版在 v5 基础上，收敛用户 07-10 推翻并重校的方向：管理员本质、单一必然 BE、收束句、灯的含义、双侧逃避镜像、"正确决定"读法、以及**回声机制的本质升级**。v5 仍保留作修订史。
> **v6 相对 v5 的关键改写**：① 管理员本质精炼为「你逃走时没能跟着走、留下来接住人生碎片的那个自己」＝"曾经救过你的逃避"；② 推翻双结局并列 → 单一必然 BE，v5 甲仅留"短暂挣扎"过渡阶段；③ 收束句改为「你终于不用等我了」（保灯意象、非导师口吻）；④ 撤销"灯＝生命余量"提案，灯重定位为"你自己不肯灭的光外化成他"；⑤ 管理员＝有私心/舍不得/被迫成熟仍带稚气的自己；⑥ 双侧逃避镜像（玩家用旧痛逃现在，管理员用旧暖逃分离）；⑦ "正确决定"重定义（非更明白，是在偏心里仍选不放手）；⑧ **回声机制升级**：回声是你自己当年说过却不敢想起的真话/旧自己，原样重放；⑨ 狂欢重调为"他也在逃分离"的镜像狂欢；⑩ 终章心脏瞬间＝"听见自己的声音＝逃避回声"。
> **v6 补充（2026-07-10 15:25）**：在 v6 定稿上叠加三项不动核心的强化——① 逾期通知升级为主题符号，新增「误解→怀疑→揭示」三阶段认知弧（夜 D 废稿细节：管理员原写"你最近还好吗／回来看看吧"划掉只留"你的书逾期了"，坐实"很久没见你的自己"）；② 终章新增「最后一次通知」beat：玩家期待通知却首次落空，到馆发现桌上已写好未寄出的《你的归还期限·已到期》，管理员"因为这次你自己来了"——直接落地"他爱你所以放手／终章停寄通知"；③ 管理员定位用语纪律："帮助成长"一律改为"陪你走完最后一段路"（陪伴非上下位）。

**核心定义（2026-07-10 用户重述，一句话锚定）**：
> 《逾期之书》＝一个被现实磨损的人，在午夜图书馆里重新遇见那个曾经替自己保存人生碎片的自己，并最终允许他停止等待。

## 一、不可动摇的设计原则（北极星与纪律）

1. **脊柱（不可换）**：图书管理员 ＝ 逾期最久的那本书 ＝ 玩家自己遗落的版本。莫比乌斯环：书是遗忘的碎片，馆员是留下的自己。此真相属后期揭晓（夜 D 闸门 → 终章），前期只立馆员人设与「还书循环」，绝不提前说破。
2. **管理员本质（v6 重写）**：他不是"不愿成长的自己"，而是**你逃走时没能跟着走、留下来接住人生碎片的那个自己**——即"曾经救过你的逃避"。区别关键：他不是"道德上更明白的自己"，是"资源耗尽时替你保管住碎片的自己"。夜 C「不急，下次还」＝「那时候的你还没准备好」，非纵容。
3. **表层 decoy**：前期每一夜让玩家以为在「帮别人还书 / 解送错通知的谜」，先不引到自己身上；第二、三幕回收成「其实是你」。decoy 命门＝玩家前期必须读成「别人的事」。
4. **北极星**：To the Moon 式情绪叙事优先。无惩罚式失败——每夜都通向一个结果（归档 / 归还 / 私藏 / 隐秘），非通关 / 失败。
5. **BE 温度（最高指令）**：落点非纯惨纯痛，而是「真实、温馨、所有人竭尽全力做到无憾、但只能到这里」。限度温柔接受，不滑向为悲而悲。任何痛都必须落在这句话上。
6. **逃避引擎（机制核心，仅设计层命名）**：每次「不在场」动作当下有效（即时止痛 → 成瘾 → 循环成立），但把自己「对（倒）下」 → 遗落一个版本的自己 → 复利累积成破碎。玩家重复做出「当年习惯的动作」，当下利好＝伪 HE；累积到揭晓才生反差。这不是正向改变 / 救赎故事——弧光是「对照」而非「净化」。**纪律：此词仅用于设计文档与引擎命名；玩家可见 prose 尽量少出现「逃避」二字，改从「你那时只会这个」写，避免玩家形成道德判断。**
7. **认知冲击 ≠ 事件冲击**：反对加事件炸弹（惨剧 / 死亡 / 背叛，会击穿 BE 温度 + 致郁否定）。靠① 落差（前期越轻日常 → 揭示越重）② 渐进拼合（多线索 → 玩家亲口说「是我」）③ 限度温柔落点（「只能到这里」因所有人都尽力了反而最重）。校准样本《海边的曼彻斯特》。
8. **管理员行为边界（写死）**：只陪伴 ＋ 寄逾期通知，零说教 / 零强提示 / 零道德高位。把你唤回的正是他寄的通知（出于爱、也成锁链）。终章分离＝他停寄通知 / 你不再应答。管理员非道德优胜者——他与玩家是同一个人，都只是累了；「好」是陪伴用心非道德高位。
9. **题眼·感谢而非战胜**：管理员不是需要被消灭的阴影人格，而是**保护机制**——是你当年唯一会用的求生方式，留下的那个你。终章「离开」不是战胜他，而是**感谢他，然后自己走**。这避开「过去的我受伤了 → 治愈 → 完整」的俗套；离开＝感谢，情绪级别高于战胜。
10. **管理员＝有私心 / 舍不得 / 被迫成熟仍带稚气的自己（v6 核心）**：最动人处不是"知道何时放手"，是"知道该放却舍不得"。写成完美理性过去自我 → 成长寓言（任务完成你该走）；写成真人 → "我知道你该走，但今晚再坐一会儿好吗"才痛。呼应已锁"管理员非道德优胜者，与玩家同一人、都只是累了"。**护栏**：私心方向若不慎会让管理员过萌 / 过 relatable 失"非人"诡异感（他是书 / 构造），须留一丝非人（私心该像缠留非甜宠），否则泄脊柱"他非人是你裂开的一块"。
11. **双侧逃避镜像（v6 结构升级）**：玩家用旧痛逃现在，管理员用旧暖逃分离——同一种困境两人各逃一边，管理员不再居"更明白"位。玩家逃避为主、管理员逃避为镜，权重不可对等否则成寓言。
12. **"正确决定"重定义（v6）**：管理员非"更明白的自己"，是在满身偏心里仍选了**不放手（放你走）**；决定之"正确"恰由"不理性（偏心）"反衬——正确的不是他，是他在舍不得中仍做此决定。
13. **自认红线**：A/B/C 绝不点破「其实是你」，否则 decoy 命门塌、莫比乌斯环失效。自认只发生在夜 D 闸门，且由玩家自己合拢（拼图合拢，不是被告知）。
14. **区域 ≠ 线索一一对应**：区域只是线索安放的地方，须与图书馆场景适配、带对应展示隐喻；故事由多线索跨区互动触发，网状散布，非沿区域线性推进。已锁 6 基区 ＋ 2 管理员区共 8 区不可换，「永远打不开的空间」只能设为第 9 个 meta 空间，不碰任何功能区。
15. **单一必然 BE（v6 推翻 v5 双结局并列）**：终章落点只有"告别过去、带着舍不得的自己走"一条必然线。v5 甲（留下 / 困守）不整段删，降级为**短暂挣扎过渡阶段**：玩家说"我不走" → 管理员说好 → 翌夜玩家自己察觉他正在消逝 → 终于肯走。它证明"留下"也曾是真的渴望，但必然 BE 不在此停。
16. **灯的含义（v6 重定位）**：撤销"灯＝生命余量"提案（非生和死的关系）。灯从来不是管理员的，是**你自己不肯灭的光外化成他**——你那份对自己的舍不得，长成了馆里那盏一直亮着的灯。保意象、不绑生死、接"带着舍不得的自己一起离开"。
17. **终章 callback 落点**：第一幕夜 A/B/C「正确做法」闪回落在「知 ≠ 行」缝隙，不写成全做到（防软化 BE）；用夜 F 三物证触发。夜 B 专属「理解父母 → 放下傲慢 → 和解」留后续，callback 只到「知」的承认。
18. **防御纹理纪律**：A/B/C 在*设计层*都是「不在场」，但*人物层*必须各有纹理：A＝用正确流程规避真实风险；B＝用「不麻烦别人的孩子」维持关系；C＝用责任感包装自我麻木；管理员＝用等待解决。同根（不让真实脆弱的自己被看见），描写各异。

19. **[v6 补充 · 管理员定位用语纪律]**：写作者后续所有文档，将「管理员帮助玩家成长」一律改为「管理员**陪玩家走完最后一段路**」。**区别**：「帮助」隐含上下位（他懂、你不懂，他拉你一把）；「陪伴」是两个自己并肩——他不是来教你的，他只是那个留下来的你，陪你走到这儿。与原则⑧（零说教／非道德优胜者）、原则⑨（感谢而非战胜）同线：他从未居上位，落点是"并肩走到尽头、然后各自退场"。

## 二、逃避引擎（v6 · 回声机制本质升级）

```
图书馆触发：现实里遇到与"已逃避过"高度相似的处境 → 图书馆作为重发的通知单出现
   ↓
[v6 升级·回声的本质]
   那些"回声 / 重演"的内容，其实是你自己当年曾经说出口、或曾经写过、
   却后来忘了、也不敢再去想起的真话，与旧版本的自己。
   你曾经比现在诚实——某年某刻你其实说过那句真的、做过那个想靠近的动作，
   然后你把它忘了，或不敢再想起。
   图书馆不是凭空造一个新处境，而是把你自己的旧声音原样还给你，
   让你在相似处境下，再次听见自己当年那句话。
   → "回声"得以成立：你逃避的从来不是别人的故事，
     是你自己曾说过、却不敢认下的那部分自己。
   ↓
努力动作（确认：玩家 genuine 挣扎 / 冲动想做对的——摸到笔、翻到号码、鼻子发酸）
   ↓
习惯的"不在场"动作＝走捷径的完成方式（把事"做完"了，但绕开真正难的那一下 → 当下止痛＝伪HE）
   ↓
遗落一个版本的自己（碎片）
   ↓
状态钩子：玩家在该夜的轻微选择 → 改变馆内物证保存状态 / 管理员关系微进（非分支、无惩罚）
   ↓
管理员功能互动 ＋ 空间暗示（馆内物证跨夜累积）
```
- 「不在场」≠ 做错事。是「用看起来负责的方式，绕开了情感核心」。
- 努力动作必须存在。没有挣扎就直接「不在场」，玩家会觉得自己「本来就是烂人」，失去对照意义。
- 伪 HE＝走捷径完成后的即时轻松。前期真轻松，后期轻松变薄；二、三幕靠「轻松变薄、觉悟变厚」驱动。
- **回声对 decoy 的改写**：第一幕"别人的书"里的信 / 相册 / 旧事，其实都是玩家自己当年的声音碎片（玩家曾说过 / 写过，忘了）。前期读成别人，后期回照"那分明是我自己说过的话"。镜子要够像别人、够不像自己，否则提前泄底。
- **回声对终章的改写**：终章心脏瞬间"听见管理员的声音＝自己的声音"，正是此回声落地——他用的腔调，是玩家当年"我再忙一天就好"的逃避回声，外化成管理员的声音。玩家撞见"自己的声音"＝撞击点。

## 二·补 逾期通知三阶段认知弧（主题符号化 · 2026-07-10 补）

> 原设计：每晚收到逾期通知 → 回图书馆 → 探索故事。**升级**：每晚收到的是**管理员写下的逾期通知** → 回到**他为你保留下来的空间**。即"通知"从剧情入口升为主题符号：每张单都是他没敢直接说的话，借"逾期"这个正当理由，把"我想你回来"寄给你。对应原则⑧（锁链）与原则⑦（渐进拼合）。

- **第一阶段 · 误解（系统期）**
  - 玩家认为：这是图书馆系统在提醒我；"这个系统还挺执着"。
  - 管理员台词照旧："你的书又逾期了。"（玩家读成机制口吻，无个人温度）。
  - 设计落点：通知此时只是循环入口，玩家不疑有他。

- **第二阶段 · 怀疑（巧合期）**
  - 设计落点：**通知的出现时机，与玩家刚经历的事太巧**。
  - 夜 A：玩家刚翻到那封信 → 第二天收到《未寄出的信》逾期提醒。
  - 夜 B：玩家与家庭产生连接（翻相册书 / 发红包）→ 收到《家的照片》逾期提醒。
  - 夜 C：玩家刚处理完漏雨旧事书 → 收到相关逾期提醒。
  - 玩家内心：从"系统真执着"滑到"……这个系统是不是认识我？"
  - **纪律**：此阶段仍不揭底；玩家只是起疑，管理员不接话、不解释。

- **第三阶段 · 揭示（夜 D）**
  - 设计落点：夜 D 闸门，玩家在管理员桌上发现一叠**废稿**——
    > 你最近还好吗？
    > ↓（划掉）
    > 回来看看吧。
    > ↓（划掉）
    > 你的书逾期了。
  - 最后只留下最普通的一句"你的书逾期了"——因为这是他**唯一敢说的话**。那些"系统提醒"全是管理员写的；他不曾说"我想你回来"，只敢借逾期的名义把你唤回。
  - **符号意义**：此细节坐实 v6 管理员定位——"不是神、不是心理医生，只是一个很久没见你的自己"。他与玩家的所有对话，从来只有"逾期"这一种被允许的靠近方式。

## 三、序章·续借（教学关）

> 序章通知先以"系统重发"面貌出现（玩家误解期），为上面三阶段弧的第一阶段埋种。

- **馆内处境**：你收到一张逾期通知单（你以为是新发的，其实是图书馆按老习惯重发的——**后期才懂，是管理员写来唤你回来的**）。手里有一本迟迟没还的书。管理员温和提醒：「可以续借，或者——下次来再还也行。」
- **努力动作**：你今天其实有空，可以现在去还。你犹豫了一下，书都掏出来半本。
- **习惯的「不在场」（走捷径完成）**：你说「下次吧」，把书塞回包里，办了续借。通知单「处理掉了」，书还在你这儿。最小单元：用「延后 / 续借」替代「真正了结」。
- **管理员功能互动**：他熟练地帮你办续借，动作温柔。你瞥见他桌上也压着一摞没还的书（种子：他也是逾期者 / 他＝你）。
- **遗落碎片**：无（教学），但种下「管理员的温柔＝纵容」的种子——后期才懂，他每一次「不急」「下次吧」，都是在替你留后路，也替自己留你。
- **设计意图**：体验一次最小循环（通知单 → 回声 → 续借 → 轻松 → 未了结），且立刻确立「管理员强引导继续逃避」是图书馆的底层温柔。

## 四、第一幕·碎片（三域，表层 decoy 沉浸期，全在图书馆内）

> decoy 保留：每夜你先以为处理的是「别人的遗物 / 别人的事」，后期回照成「是你自己的碎片」。**v6 升级**：这些"别人的声音"是你自己当年说过、却忘了也不敢想起的真话（回声）。每夜附「自认播种」四项 ＋ 状态钩子。

### 夜 A《夹在书里的信》— 关系域（没说出口的话）

- **馆内处境（回声）**：你还书时，一本书里掉出一封没署名的信。管理员让你处理。你下意识当它是别的读者落的。
- **回声的真相（v6 升级）**：你翻开信，写的是对某个人没说出口的、很重的话——那其实是你自己**当年曾经写过 / 说过**的，只是后来你忘了，也不敢再去想起。看到某一句时，你突然记忆闪回：当年你也有这样一封信 / 这样一个人，你当时也是「多一事不如少一事」，把话咽了回去，假装没看见。闪回不揭底，只让你自己心里一沉——「我当初就是这么绕开的，可那句话，曾经是我自己说过的」。
- **努力动作（确认）**：闪回过后，你冲动地想：也许我该对生命里那个人，也写点真的。你摸到笔，在便签上写了两句真话，手在抖。
- **习惯的「不在场」（走捷径完成）**：你把便签揉了，把信按规矩贴标签放进失物招领信箱（任务「完成」了）。你秉持「多一事不如少一事」——不是接着去解决，而是送往信箱了事。你「做了该做的」，当下轻松。但真话没寄给该寄的人，也没说出口。**关系域纹理＝用正确流程规避真实风险**（把「表达」替换成「处理」）。
- **正确答案（当年没做）**：当年你该对那个人说出 / 写出那句真的重话，而不是让关系停在表面和平。
- **管理员功能互动 ＋ 暗示**：他递你胶水和便签（功能互动：帮你贴失物标签），随口说「很多人把想说的话，先写在别的地方」。你没接。后来你在服务台抽屉发现你揉掉的便签被收着了（跨夜伏笔①）。
- **遗落碎片**：一张没寄出的真话便签 → 遗落「表达版自己」。
- **[状态钩子]（非分支）**：在「写便签」环节给玩家一个轻选择，只改馆内状态——
  - 写一句真话（如「其实我很想你」）→ 管理员把便签摆在抽屉「最上层、压平」的位置，终章可见其被郑重收着；
  - 写一句安全的话（如「最近还好吗」）→ 管理员把它夹进登记册里，位置中性；
  - 什么都不写、直接送信箱 → 管理员替你补一张「未署名」标签，仍收着。
  - 三种都不惩罚、都通向同一结果；差异仅在终章 callback 时玩家看到的「这些不是你做错的选择，而是你每一次保护自己的方式」的具体物证形态。
- **自认播种（夜 A）**：① 借阅人墨团——信所属那本书借阅人栏涂烂、裁角，墨团边缘隐约有手写「下次」；② 镜面映射——信里人「多一事不如少一事把话咽了」严丝合缝对应你的关系域纹理（叙述角度放「别人的故事」）；③ 玩家是常量——你刚把伞忘在门廊，下一本「别人」的书里就有人在门廊留伞（呼应极轻）；④ 管理员暗线——他记得你揉掉的便签被收着（伏笔①），前期读成职业习惯。

### 夜 B《满员的书架》— 家庭域（被疏忽的关系）

> **家庭内部关系的疏忽——底层动机（本夜核心，写深刻）**：
> - 「让人省心的孩子」是察言观色练出来的，不是天生的。你是家里「最让人放心」的那个：读书出去、留在大城市、从不惹事、报喜不报忧。但你不是生来就这样——小时候的家，情绪是会突然变天的：父母也会因为钱、因为累、因为各自没处安放的委屈吵起来，那股火有时会莫名落到你头上。你很小就学会了看脸色，把自己的需要、怕、委屈都收起来。不是家里不爱你，是那时的家承接不住一个孩子的情绪——他们自己都在被生活推着走。于是你心里落下一句话：真实的情绪，在家里是「不安全」的。报喜不报忧，就是从这儿长出来的自我保护。
> - 他们的关心是真的，只是和这个时代脱节了。催你考公、劝你找个稳定工作、总担心你一个人在外面吃不好睡不好、一回家就催婚催生。这些话你听着烦、觉得被冒犯，可在他们那代人的理解里，那就是「最好的爱」。他们不是不在乎你心里怎么想，是他们真的不懂——这个信息爆炸、变得太快的时代，孩子心里在经历什么，没人教过他们；他们也是带着过时的知识、第一次做父母、第一次走到今天。错位，不等于恶意。（这一层，此刻的你还看不太清、也不太愿意看——你只觉得被冒犯。真正读懂它、放下这份年轻人的傲慢，是后来的事。）
> - 于是形成了「互相回避的安静」——两边都在小心保护对方收到的那个版本。家里有事瞒着你；你有事也不说。两边都以为自己在「为对方好」，结果谁也没真正靠近谁。这不是谁对谁错，是关系上的长期疏忽——把彼此推到「报平安」的安全区。而这一夜真正被聚焦的，是你自己：是你，为了图这份「平稳」，主动选择了不再靠近。
> - 具体的疏忽事件：有一年父亲生过一场病，家里人瞒着你没说。你后来偶然知道，第一反应不是买票回去，而是多打了笔钱、在家族群发「注意身体、别省着」。你本可以回去看看，但你没有。你不是忙，你是已经不太会用「人」去靠近家人了——只会用「钱」和「报喜」去替代「在场」。
> - 为什么不回家（「忙」只是挡箭牌）：回老家对你早已不是轻松的事。回去要面对饭桌上那些错位的关心；你想说一句「我其实挺累、那年爸生病我后怕」，可话一出口，大概率换来的是他们那套旧地图的规劝，于是你又咽了回去。你和他们之间，仿佛隔着一整个时代——明明最亲，却无话可说。再加上这些年用钱替代陪伴攒下的亏欠，和父母真的老了。「忙」只是借口。真正让你不回去的，是回去就要面对这段被你疏忽的关系，就要重新去接那份你早已不知怎么接的、错位的亲密。所以转账、红包、报平安，是你给得出来、代价又最小的那点「连接」。

- **馆内处境（回声）**：一片「家庭相册书目」区要归架。你翻到一本旧相册书，是你小时候的家——某页是年夜饭的桌，桌边空着一个你常坐的位置。你下意识把它当别的人家 / 别的读者的遗落物。
- **回声的真相（v6 升级）**：你曾经，其实是想回去的。某年某次，你甚至把那个号码按到一半、或写好了「我想你们」又删掉——那句想靠近的话，是你自己当年真说出口过（或写到一半）的，只是后来你用"忙"把它盖过去了，也慢慢不敢再想起自己曾那么想靠近。相册书里那家人的空位，照的正是你当年那句没递出去的「我想你们」。
- **努力动作（确认）**：手指停在那页。你其实想过，今年该回去一趟，或至少打个长的电话、说句「我想你们了 / 那年爸生病，我其实很后悔没回来」。你拿起手机，翻到那个存了很久却很少拨的号码，指腹悬在拨通键上——可你脑子里已经预演了一遍：接通后，那句「我想你们」大概率接不住，会变成「吃饭了没 / 工作忙不忙」，然后你们又安然滑回各自的剧本。你知道，你怕的不是拨号，是拨通后依然无话可说。
- **习惯的「不在场」（走捷径完成）**：你没拨。只发了个红包，附一句「最近的都好，勿念」就锁屏，书归架了，整理「完成」了。你用「尽了责」的完成感，替代了真正的回去、真正的对话。**家庭域纹理＝用「不麻烦别人的孩子」维持关系**（用尽责 / 红包替代在场，用报喜替代说真话）。（同根：在家庭域，不在场长成「疏忽关系、用钱替代陪伴」；在自我域，长成「用做完盖情绪」。同一种本能，两个面：都不让真实脆弱的自己被看见。）
- **正确答案（当年没做）**：当年你该真的回去，或真说「我想你们 / 那年我没回来，我后悔」，而不是用钱和报喜替代在场、用省事替代面对。
- **管理员功能互动 ＋ 暗示**：他递你书立和除尘布（功能互动），你说「马上好」却很快做完。后来你发现那本相册书被他单独摆在「待你再来看」的架位（跨夜伏笔②）——他记得你翻得最久的那页，也记得那个空着的位置。
- **遗落碎片**：一个没拨出的号码 / 一句没说的「我想你们」 → 遗落「归家版自己」。
- **[状态钩子]（非分支）**：在「发红包 / 拨号」环节给玩家轻选择，只改馆内状态——
  - 悬在拨通键很久才放下、只发红包 → 管理员把相册书空位页折了个角，摆在「待你再来看」架位最显眼处；
  - 直接发红包没犹豫 → 相册书正常归架，但他另拿一张便条写「这页他看得最久」压在书里；
  - 多打了一笔钱（沿用动机块「用钱替代」）→ 他在书脊贴一枚小星标，暗示「这次又用钱绕过了」。
  - 三种都通向同一结果；差异仅在终章时玩家看到的「保护自己的方式」物证。
- **自认播种（夜 B）**：① 借阅人墨团——相册书借阅人栏同样涂烂、边缘「下次」；② 镜面映射——相册书里「年夜饭空位 ＋ 只发红包」对应你的家庭域纹理；③ 玩家是常量——你刚发完红包，书里那家人也只发红包；④ 管理员暗线——他记得你翻最久的空位页（伏笔②）。

### 夜 C《雨水收集器》— 自我域（咽下的情绪）

> **「用忙碌麻痹难过」是怎么来的（前置记忆，本夜闪回讲清）——同一时期的多重打击**：
> - 那段日子：一边是家里长辈离世，你没能赶回去见最后一面（正扣夜 B：那阵「忙」也把你从家拽走了）；一边是工作上遇到一个很烂的领导，天天施压、甩锅、让你怀疑自己。两件事叠在同一时期砸下来。
> - 你当时怎么做：你逞强——不是摆烂，是更用力地「把事做完」。长辈的后事、领导的交付，你一件件扛下来，做完一件划掉一件，心里暗暗相信「只要把这些都做完了，我就好了」。你确实把「事」都解决了：葬礼办妥、项目交付。但你没发现自己只解决了事，没解决自己——那股被压进去的疼，你一天也没碰过。
> - 机制落成：你发现「只要手不停、把事做完，心就不那么疼」。从那以后，任何难过袭来，第一反应不是感受，是找事做。这就是「忙碌麻痹」的来历——它诞生于一次「做完 ＝ 痊愈」的错觉。

- **馆内处境（回声）**：馆里一处老漏雨，管理员放了个旧搪瓷盆接水，说「这地方老漏，习惯了」。你被叫去帮忙处理——漏雨引发的事一件接一件：先要接水、要挪书、要擦干净。此外，这次你也翻到一本「漏雨旧事书」（别人的遗落物），借阅人栏同样涂烂。
- **实在的对比**：雨水声让你疼了——你闪回那段多重打击的日子：当年你也是这么「处理」难过的——用一件接一件的事把自己填满，以为做完就好。这一次，漏雨又给了你同样的「事」：接水、挪书、擦地。你本能地扑上去做，和当年一模一样。你甚至比当年更熟练了。
- **漏水对应说明**：漏雨＝你一直没补好的那道情绪裂缝（长辈离世 ＋ 烂领导的疼，从没真正面对）；接水 / 挪书 / 擦干净＝你忙着用「小事」不让情绪上来，把裂缝渗出来的水一遍遍擦掉，却从没想过把裂缝本身补上。你处理漏雨的方式，和处理难过的方式，是同一个动作：解决表层，绕开根源。
- **努力动作（确认）**：搬书时鼻子发酸，你几乎要停下来，跟管理员说点什么——或者干脆坐下承认「我很难过，我其实没好」。
- **习惯的「不在场」（走捷径完成）**：你抹了把脸，把最后几本书码齐，跟他说「好了，没事」，然后去泡了杯咖啡、把灯调亮，用「把事做完 ＋ 一点自我安抚」的完成感盖过了那阵情绪。你「处理」了漏雨，也「处理」了心情——都用流程。**自我域纹理＝用责任感包装自我麻木**（用做完 ＋ 安抚压住真实情绪）。（同根：从「忙」里学会了把家推到以后，也学会了把情绪推到「做完再说」。）
- **正确答案（当年没做）**：当年你该允许自己停下来，承认「我很难过，我其实没好」，而不是用完成感和一点安抚把情绪咽下去。
- **管理员功能互动 ＋ 暗示**：他递你毛巾（功能互动），没多问。后来你发现那盆被他每晚摆在老位置，盆边你那天放下的东西也在（跨夜伏笔③，物证持续存在）。
- **遗落碎片**：一阵没被承认的难过 → 遗落「感受版自己」。
- **[状态钩子]（非分支）**：在「抹脸说没事 / 停下承认」环节给玩家轻选择，只改馆内状态——
  - 几乎停下、最终还是说「好了没事」→ 管理员把盆边你放下的物收进一个小盒，标「雨天物」；
  - 真说出半句「我其实不太行」又咽回去 → 他将那半句记在逾期通知背面（手改痕迹），你后来才看懂；
  - 完全用咖啡 ＋ 调灯盖过 → 他在灯下多摆了一把空椅子，像是知道你本该坐下来。
  - 三种都通向同一结果；差异仅在终章三物证合流时的形态。
- **自认播种（夜 C）**：① 借阅人墨团——漏雨旧事书借阅人栏涂烂、边缘「下次」；② 镜面映射——旧事书里「用一件接一件的事填满、以为做完就好」对应你的自我域纹理；③ 玩家是常量——你刚「做完盖情绪」，书里主人公同款动作；④ 管理员暗线——他记得你放盆边的物（伏笔③）。

## 五、第二幕·管理员之夜（夜 D/E 逼近「你就是我」，夜 F 预兆）

> 本幕不再空流程：靠「你和管理员越来越像」的寒意推进，直至温柔者反常赶人。

### 夜 D《背后楼梯》— 你就是我（闸门 · 双爆点）

- **馆内处境**：管理员第一次带你进管理员专属区·背后楼梯 / 档案室（第二幕解锁）。说「这边有点乱，帮我理理。」——他其实在给你看他的世界。
- **故事（逼近暗喻）**：
  - 你发现他桌上也压着一本「逾期最久」的书，封皮被摩挲得发毛，和你序章那本一样旧。
  - 他讲：「我也在等一个人回来还书，等了很久，那人总说下次。」你听着，心里一种说不清的熟悉感慢慢浮上来——像在听自己的事。
  - 他做着和你一模一样的小动作：把某封信塞进抽屉说「下次处理」、把一杯茶推到「不急」的位置、被问急了就笑说「别急」。你突然意识到：管理员的每一个习惯，都是你的习惯。
- **闸门（逻辑层·显式机关）**：他让你帮忙理档案，你翻开那本「逾期最久的书」的借阅卡——**签名是你，借出日期正是序章续借那天**。这一刻，前面三夜埋下的种子同时回照：三本涂烂的借阅人栏、墨团边缘的「下次」是你写的；「别人」的故事全是自己的事；管理员只对「你」熟。你不是被谁宣布，而是自己「啊，原来一直是我」——拼图合拢。对应核心原则：破碎是玩家自己数出来的。
- **[情感层爆点]（不抢释放戏，埋入夜 D / 终章回收）**：逻辑层答「你是谁」之后，再给玩家一层更痛的领悟——不是「管理员困住了我」，而是：
  > 不是管理员困住了我。是我（当年的不在场）需要有人留下，于是那个没走的我，一直留在了这儿。
  
  他不是我「创造」出来的东西，他本就是我自己留下的那部分——那个在我逃走时没能跟着走、留下来接住人生碎片的自己。同一性不破——呼应题眼「他不是阴影人格，是保护机制」。此爆点若夜 D 不展开，可延后到终章揭晓时释放；但种子（「他每个习惯都是我的习惯」「他也在等一个人回来」）必须夜 D 种下。
- **作用**：decoy 彻底裂向「你」；逻辑层 ＋ 情感层双爆点蓄满夜 E 赶人的反差。这一拍只答「你是谁 / 这些书都是你 / 他本就是留下的你」，不抢释放戏。

### 夜 E《赶你出去的夜》— 一反常态（BT 式「因爱放你走」）

- **馆内处境**：某夜你照常来，管理员却反常地站在门口，没笑，挡着门。
- **一反常态**：他说「今晚你回去吧。别续借了。」——这是他第一次不温柔、第一次主动切断「不急」的回路。
- **双关**：「有些书，拖着不还，对谁都不好。」一语双关你的不在场，也双关他自己（他何尝不是「拖着不让你走」的那本逾期书）。
- **努力 / 不在场**：你错愕、想赖着不走，他用身体轻轻把你推出门：「走吧。」灯在身后亮着。你第一次被「赶」出图书馆。
- **作用（分离前奏）**：打破「管理员永远纵容」的预期。BT 隐喻落地——没逃的那个你，独自面对了留人的无力；而因爱你，他选择放手。为第三幕「他其实一直在用温柔留你、现在却推你走」的甜蜜反差蓄力。

### 夜 F《三物证合流》— 预兆夜

- 你终于在某次「不在场」后，直接问：「你到底是谁？」他不直接答，把你历夜遗落的东西摆出来——**便签、相册书、雨水盆边的物**（三个跨夜伏笔在此合流，且呈现形态即来自各夜状态钩子）——「这些都是你落下的，我替你收着。」
- 你仍可选「不在场」（把东西推回「下次整理」），但这次 Relief 几乎没了。身份揭晓的铺垫在此完成。
- **剧情同步点**：你「知道自己在绕开」这件事本身，成了新的重量。循环不再是无知地转，是清醒地转。

## 六、第三幕·最后的狂欢（夜 G/H/I… 多夜，甜得刚好蓄力又有点反常）

> 玩家被赶走后习惯性地又回来了（贪恋）。此时玩家已知「是我」，却仍贪恋被稳稳等着——管理员恢复温柔，甚至更开朗，因为他知道自己要失去了，于是把最后的甜全给够。每夜快结束都留一句「明晚你还来还书吗？」，甜蜜里拽回 BE 的线。
> **v6 升级（狂欢重调）**：这甜不是"他知将失故给甜"，而是**他也需要这段时间、他也在拖延**。玩家问"你不是说我要早点回"→他"嗯，但今天晚点也没关系吧"→玩家悟：他非在教育我，他也在逃分离。狂欢＝镜像狂欢——**双侧逃避在此同框**：玩家用旧痛逃现在，管理员用旧暖逃分离。

- **节奏**：这些夜节奏快（每夜 5–8 分钟），不拖泥带水，是「最后的狂欢」。
- **管理员状态**：开朗、话多，主动带你「参观」图书馆——「这盏灯是我装的」「那排书架是我排的」「你看，这里全是给你留的」。有点炫耀，甜得有点反常（你隐约觉得他比平时更用力）。**他也在用"多陪一夜"逃开即将到来的失去**。
- **收尾固定句（每夜）**：他笑问——「那明晚你还来还书吗？」既是邀请，也是把「续借 / 回来」这根 BE 的线轻轻拽回。

### 代表夜（写 2–3 个示意，结构可重复多夜）

- **夜 G《灯是谁装的》**：他带你看了那盏一直亮着的灯，说「这灯，是专门为你留的」。你心里甜，但闪过夜 E 他被你推出门的画面——甜里有刺。→ v6 灯的含义在此点一丝：灯不是他的，是你自己不肯灭的光。
- **夜 H《档案室半开的门》（给夜 Z 埋种子）**：某一夜，狂欢正甜，你「不小心」又晃到档案室半开的门前。里面那盏灯下，你序章续借的那本旧书被单独摆在显眼处，卡片上他写的一行小字：「等你还。」你没多想——此刻你明知「是我」，却仍贪恋这份被稳稳等着的甜，把这行字当作情话般收下，没去碰「该还」那一下。→ 这就是夜 Z「你带着那本书回来」的伏笔：你一直没真正还的，是序章那本。
- **夜 I《最后一盏灯前的玩笑》**：你们像老友一样把某本难还的书推到「明天」，笑闹。但他笑到最后，目光软了一下，没接话——他在拖延自己的失去，正如你在拖延你的面对。

## 七、终章·夜 Z《带书回来的人》（单一必然 BE ＋ callback）

> 玩家选择把这一切了结，来最后一个夜晚，手里带着序章续借的那本书（夜 H 伏笔回收：你一直没真正还的，就是它）。

- **解释一切（含情感层爆点回收）**：
  - 管理员＝你（逾期最久的那本书＝你遗落的版本；他是一直留下来的那个你；那个你逃走时没能跟着走、留下来接住碎片的自己＝"曾经救过你的逃避"）。
  - 整座图书馆、那盏灯、每张逾期通知＝他留下的温柔的构造，不是欺骗，是陪伴的用心。**灯从来不是他的，是你自己不肯灭的光外化成他**。
  - 你每一次回来、还书、陪他玩＝**最后一道不在场**：你贪恋「被稳稳等着」的感觉，不敢真的往前走（门一直没锁）。
  - **[情感层落点]**：不是他困住了你。是你当年需要有人留下，于是那个没走的你，一直留在了这儿。他不是你要战胜的阴影，他是你给自己留的退路——**所以你不是战胜他，是感谢他，然后自己走**。

- **[v6·单一必然 BE 落点]**：
  - **（甲 · 短暂挣扎过渡，非终局）**：你把书放回书架，说「我不还了，我留下。」——这是真的渴望，但只是过渡：管理员说"好"，翌夜你却自己察觉他正在消逝（他本就是留下的你，你既已肯走，他没了存在理由）。你终于明白，留下＝永远停在自己不敢往前的地方。于是你还是肯走了。**甲证明"留下"曾是真的，但必然 BE 不在此停。**
  - **（乙 · 告别过去 · 必然落点）**：你把书真正还给他、插回原位，说「该走了」。门没锁，你走出，灯留原地。各夜当年没做的（说出真话 / 回去 / 承认难过）在此刻汇成「往前走」这一个动作。你不是痊愈——是带着「知道却曾做不到」的自己，肯往前走。

- **[v6·终章心脏瞬间 · 回声落地]**：在狂欢收尾、你首度在馆内轻快提外间明日计划、自然拉门停住时——管理员无脑接老套「再坐会儿茶温着 / 书没归位明晚再走」。你留下。你忽然听见他声音的腔调，和你当年「我再忙一天就好」**一模一样**。你顿悟：他早非保护，是替你表演你自己的逃避；你开始把馆当生活、打算留下（出于舒服非必要）。最珍惜的人之形，正被他的爱把你变成他的样子（永远等 / 不敢走 / 暖意当壳）。故终章那下推＝**「我不能再爱你爱到把你变成我的样子」**——非职责、非"你准备好了"，是带全副舍不得的真放行。（这正是 v6 回声机制的终极落点：你听见的"他的声音"，是你自己当年逃避的回声。）

- **收束句（v6 升级 · 保灯意象 ＋ 感谢口径 ＋ 非导师口吻）**：
  > 「灯还亮着。——你终于不用等我了。」

  「你终于不用等我了」是两个自己之间的告别，非导师口吻；"灯还亮着"保留灯＝你自己的光的意象。感谢口径仍立：离开是感谢（他接住过你），不是战胜。

- **[v6 补充 · 最后一次通知 · 强化回收]**：在终章抉择之前，落一个专门 beat，把"他爱你所以放手"从台词变成动作——
  - **循环惯性**：玩家一直期待"今晚会收到什么通知？"——整个游戏循环都是管理员主动把玩家拉回来。
  - **第一次落空**：这一夜，没有通知。玩家照旧来到图书馆，发现管理员桌上**有一张已经写好、却没有寄出去的通知**：
    > 《你的归还期限》
    > 已到期。
  - 玩家问："为什么没有发？"管理员答："因为这次……你自己来了。"
  - **为什么重要**：此前每一次回归，都是他主动伸手；最后一次，他选择**不伸手**。这比说"你该走了"更贴 v6 想表达的——**他爱你，所以放手**。也直接落地原则⑧"终章分离＝他停寄通知"。这张未寄出的单，是整部作品最长的一句"我想你回来"，也是第一次他忍住没寄。
  - **放置位置**：在 ① 解释一切 之后、③（甲短暂挣扎 / 乙告别）抉择之前；可与夜 A/B/C callback 叠层，让"未寄出的单"与三物证并置——玩家看见自己落下的碎片，也看见他写满又咽回去的那一句。

- **BE 温度（不可滑偏）**：两个版本的你都竭尽全力，只能到这里。限度温柔，非致郁；不是「你痊愈了」，是「你终于肯往前，带着这份舍不得的自己」。

### 终章 callback（第一幕 A/B/C 正确答案闪回 · 知≠行）

> **设计意图**：终章把第一幕夜 A/B/C 的「正确做法 / 成长部分」做闪回 callback，说明主角其实一直知道要怎么做——但情绪和压力不会因为他懂了道理就凭空消失。这不是救赎、不是「你痊愈了」，恰恰相反：它把「只能到这里」的限度从「不懂」改成「懂了也还是难」，反而更深、更诚实地落住了 BE 温度。
> **关键纪律**：callback 必须落在**知 ≠ 行**的缝隙，绝不能写成「现在三件都做到了」。闪回同时呈现① 努力动作（证明你**知道且想做对**）② 情绪与压力如何压过道理。落点＝「我从来都知道要怎么做，只是懂了也还是难」。各夜 callback 物证形态，直接来自该夜**状态钩子**的玩家选择（即感谢口径的物证：「这些不是你做错的选择，而是你每一次保护自己的方式」）。
> **v6 回声回扣**：闪回里那些"正确答案"，本就是玩家自己当年说过 / 写过、却忘了也不敢想起的真话（见第二节回声机制）——你不是不懂，是你曾懂、曾说出，却把自己那句话弄丢了。

- **放置位置**：在① 揭晓「管理员＝自己」之后、③ 抉择分离之前，作为通向「往前走」的「知」段落；可与② 不在场开心的夜叠层。用夜 F 已合流的三物证（便签 / 相册书 / 雨水盆边物）作触发物。
- **三封迟到的正确答案（闪回内容）**：
  - **便签 · 夜 A（关系域）**：闪回你摸到笔、写了两句真话又揉掉、送信箱了事——物证即你当夜选择留下的便签形态。旁白——「那句话，其实是我自己当年说过的。可电话拿起来，情绪先一步堵住嘴。知道该说，和说出口，中间隔着一整场怕。」
  - **相册书空位 · 夜 B（家庭域）**：闪回你手指停在年夜饭空位、号码悬在拨通键、只发了红包——物证即相册书空位页的折角 / 便条 / 星标。旁白——「那句『我想你们』，我也曾到过嘴边。可『忙』背后那股压力——怕无话可说、怕被旧地图接不住——比道理重。知道该回，和回得去，是两件事。」（此处只做「知」的承认，夜 B 专属的「理解父母 → 放下傲慢 → 和解」留后续，不在此完成。）
  - **雨水盆边物 · 夜 C（自我域）**：闪回你鼻子发酸想停下、却抹了脸说「好了没事」去泡咖啡——物证即盆边物 / 通知背面的半句 / 空椅子。旁白——「我甚至知道该承认『我很难过、我没好』。可那阵压力把『做完』焊成了本能，道理在它面前轻得像灰。知道该停，和停得下来，隔着我自己。」
- **统一落点（本段核心句）**：这三段不是「我终于做到了」，是「我终于承认——**我从来都知道要怎么做，只是情绪和压力不会因为我懂了道理就消失**」。所以那些版本（表达版 / 归家版 / 感受版自己）遗落了，不是因为我蠢、不懂，是因为懂了也还是难。这正是「只能到这里」的诚实：限度不在无知，而在生而为人那点沉甸甸、绕不开的难。
- **与乙结局衔接**：「门没锁你走出」不是「我痊愈了」，是「我带着这份『知道却曾做不到』的自己，肯往前走了」。A/B/C 当年没做的，有的已经迟了（那封信、那年爸生病），但「往前走」这一个动作，是此刻唯一还来得及的「正确答案」。

## 八、跨夜回照机制（修复「外围 / 单元剧」问题）＋ 图书馆作为角色

- **馆内物证持续存在（含状态钩子差异）**：便签（服务台抽屉，按玩家选择呈不同保存形态）、相册书（「待你再来看」架位，按选择呈折角 / 便条 / 星标）、雨水盆（每晚老位置，盆边物按选择呈盒 / 背面半句 / 空椅子）——都是图书馆空间状态，不是别人的故事。
- **通知单累积**：每张逾期单带手改痕迹（管理员手写划掉又写），后期玩家发现笔迹越来越像自己；夜 D 揭示这些单全是管理员所写（桌上废稿"你最近还好吗／回来看看吧"划掉只留"你的书逾期了"）；终章那张**已写好未寄出**的《你的归还期限·已到期》，是循环里唯一一次他忍住没寄——把"停寄通知"落成动作。
- **管理员记忆碎片**：他偶发掉落「他也是被遗忘的自己」的碎片，跨夜收集 → 终章拼图。
- **氛围随内心变**：灯色、雨声、台词密度随觉悟微调（夜 E 赶人时灯骤暗；第三幕狂欢时灯偏暖偏亮，但暖得反常）。

### [图书馆作为角色 / 探索欲]
为补「情绪线结束后中间动力不足」，把图书馆本身升为可探索的「角色」，但**绝不削弱 8 功能区的固定性**（6 基区 ＋ 2 管理员区已锁不可换）：
- **书架移位**：某些书架每日位置微调，长期观察会产生「这馆在呼吸」的直觉，驱动玩家每天回看。
- **雨天限定书**：某些书仅雨天出现（与夜 C 雨水收集器对位），晴天不可见——把天气变成探索变量。
- **馆员收通知**：某些逾期通知不是玩家收到，而是管理员收到（与机制「通知是唤你回来的锁链」对位）——玩家会好奇「谁在给他寄通知」，埋他也是「被等者」的暗线。
- **第 9 空间（ meta · 永不开启）**：管理员的来处 / 灯光的源头，设为一个门永远虚掩却推不开的房间，纯悬疑、不解锁、不解释。它**不是 8 功能区之一**，不破坏固定区；作用是给长期驱动力「我想看看这座图书馆到底是什么」，并让管理员的「留下」多一层身世留白——他不只等你，他自己也是某场未还之书留下的。

## 九、自认机制（播种 → 回收）

> 自认机制是两条回收的总闸：① 补全角色最终性格走向（从「无意识绕开」转轴到「清醒地转」）；② 让第一幕所有铺垫（墨团 / 镜面 / 玩家常量 / 管理员暗线）在 D/F/Z 一次性回照合拢。分寸：不「明显提示」——A/B/C 绝不点破「其实是你」；但要让玩家自己拼出「是我」，靠可回照的物证种子，而非 D 幕空降反转。自认是情绪上的诡异熟悉（uncanny）：慢慢觉得不对劲、最后被戳穿，不是逻辑推理出「是我」。它由玩家在夜 D 闸门自己合拢。

**写作纪律**：
- ② 镜面「够像别人、够不像自己」——细节可重合，但情绪 payload 不提前给（夜 A 只闪回「我当初就是这么绕开的」一沉，不展开；夜 B/C 同理只到「一沉」为止）。
- ③ 的呼应要轻到玩家可忽略（伞 / 红包 / 做完都是随手动作），重了就泄底。
- ④ 前期必须读成职业习惯，不能让管理员流露出「他在盯你」。
- **[v6·回声纪律]** ② 镜面映射的"别人故事"，在伏笔层可埋一丝"这其实是你自己说过的话"的回声感（如信的措辞与玩家闪回里"当年那句"同源），但前期仍须读成别人的遗憾；回照在夜 D/F/Z 一次合拢。
- 各夜「状态钩子」的物证形态，必须在 A/B/C 正文 weave 进，否则夜 D/F/Z 合流会显得空降。

### 9.2 第二层 · 夜 D 闸门（双爆点，靠种子命中注定）

- **逻辑层机关**：档案室那本「逾期最久的书」借阅卡——签名是你，借出日＝序章续借那天。
- **情感层机关**：借阅卡之外，由「他每个习惯都是我的习惯」「他也在等一个人回来」的种子，推到「不是他困住我，是我让他留下的 / 他本就是我自己留下的那部分（那个我逃走时没能跟着走、留下来接住碎片的自己）」。情感层可夜 D 种、终章释放，但不得用「创造」措辞（保持「他＝我」同一性，不重新拉开距离）。
- **合拢方式**：不是管理员宣布「你就是借阅人」，而是玩家自己「啊，原来一直是我」——拼图合拢，不是被告知。对应核心原则：破碎是玩家自己数出来的。
- **闸门纪律**：这一拍只答「你是谁 / 这些书都是你 / 他本就是留下的你」，不抢释放戏；夜 E 赶人、终章 callback 才分别接「放你走」与「知 ≠ 行」。

### 9.3 第三层 · 角色弧转轴 ＋ 铺垫回收总闸

### 9.4 给写作者的密度检查（防过埋 / 防欠埋）

- 每夜**至少埋 ① 借阅人墨团 ＋ ④ 管理员暗线**两条常量（最便宜、最隐蔽，几乎零成本）；
- ② 镜面映射每夜一条（A 关系 / B 家庭 / C 自我，已锁域分配与纹理命名，不增不减）；v6 可埋一丝"回声同源"但不提前揭底；
- ③ 玩家常量呼应每夜一条（可极轻，玩家忽略也不影响）；
- 状态钩子物证每夜 weave 进正文（便签收着形态 / 相册书折角 / 盆边物盒），不可只写在播种块；
- **红线**：任何一夜间接出现「这好像是我」「你就是本人」之类的点破句＝违规，打回重写。自认只能发生在夜 D 闸门，且由玩家自己合拢。
- **标志物候选（不锁定）**：借阅人墨团里的手写「下次」可提炼为贯穿标志物（与逾期通知卡同源），待夜 A–C 文字自然浮现后再定。

## 十、本版与 v5 的差异速查（备查）

| 项 | v5 | v6 |
|---|---|---|
| 管理员本质 | "没逃、留下扛的那个你" | **"你逃走时没能跟着走、留下来接住碎片的自己"＝曾经救过你的逃避**（非不愿成长） |
| 结局结构 | 双结局并列（甲留下 / 乙告别） | **单一必然 BE**；甲仅留"短暂挣扎"过渡（玩家说不走→管理员说好→翌夜觉其消逝→肯走） |
| 收束句 | 「灯还亮着…谢谢你一直替我守着…你得自己往前走」 | **「灯还亮着。——你终于不用等我了。」**（保灯意象 ＋ 感谢口径 ＋ 非导师口吻） |
| 灯的含义 | "温柔的构造"（未定生死） | **撤销"灯＝生命余量"；灯＝你自己不肯灭的光外化成他**（不绑生死） |
| 管理员人设 | 陪伴用心、非道德优胜者 | ＋**有私心 / 舍不得 / 被迫成熟仍带稚气的自己**；留一丝非人诡异感（护栏） |
| 逃避镜像 | 玩家单方逃避为主 | **双侧镜像**：玩家用旧痛逃现在，管理员用旧暖逃分离（权重不等，否则成寓言） |
| "正确决定" | 未专门定义 | **非更明白，是在偏心里仍选不放手（放你走），"正确"由不理性反衬** |
| 回声机制 | "相似处境重演" | **升级：回声是你自己当年说过却不敢想起的真话 / 旧自己，原样重放**（decoy＝你自己的声音碎片；终章"听见自己声音"＝此回声落地） |
| 狂欢夜 | 他知将失故给甜 | **他也在逃分离**的镜像狂欢（玩家问"不是说要早点回"→"今天晚点也没关系吧"） |
| 终章心脏瞬间 | 未定稿 | **"听见自己的声音＝逃避回声"**：他腔调＝你当年"我再忙一天就好"；推力＝「我不能再爱你爱到把你变成我的样子」 |
| 题眼 | 感谢而非战胜 | 同（保留） |
| 逾期通知（符号化） | 仅作循环入口"系统提醒" | **v6 补充：主题符号**；三阶段认知弧（误解系统→怀疑"系统认识我"→夜D揭示全由管理员所写，废稿"你最近还好吗/回来看看吧"划掉只留"你的书逾期了"）＝"很久没见你的自己" |
| 终章最后一次通知 | 无（仅"你该走了"口吻） | **v6 补充**：玩家期待通知却首次落空，到馆见桌上已写好未寄出《你的归还期限·已到期》；管理员"因为这次你自己来了"＝循环唯一一次他忍住没寄，落地"他爱你所以放手" |
