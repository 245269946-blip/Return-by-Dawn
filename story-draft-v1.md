## 框架对齐（已锁，复述以防跑偏）
- 脊柱：管理员 = 逾期最久的那本书 = 玩家自己遗落的版本（莫比乌斯环）。
- decoy：第一幕让玩家以为在"帮别人还书 / 解送错的通知"，后期回收成"其实是你"。
- 逃避引擎：每夜重复"当年逃避的习惯动作" = 当下利好（伪 HE），但每夜遗落一个版本的自己；复利成破碎。
- 正确答案：B 行动型。每夜一个玩家亲手做、以前不敢做的小动作；终章"门没锁你走出"。
- 三拍结尾：① 揭晓管理员=自己　② 相认后一夜开心逃避　③ 揭晓逃避真相 + 抉择分离。
- 跨夜累积：同一批区域反复用，线索物件跨夜持续存在、互相回照（防单元剧）。

## 第一幕：碎片（表层 decoy 沉浸期，图书馆内）

每夜"别人的书" = 一个逃避域的镜子。前期读成别人，后期回照成自己。
玩家每夜都做同一个**习惯动作**（伪 HE），正确答案则是当年没做的小动作。

### 夜 A《夹在书里的信》—— 逃避域：关系 / 未说出口的话
- **别人的书（decoy 表面）**：一本被人借走、夹着一封没寄出的信的书。逾期通知寄到了"你"手里——前期读成"寄错了，你帮忙送还"。信是某个人写给另一个人的、始终没说出口的话。
- **习惯动作（伪 HE）**：你把信原样夹回书里、把书归架——"替别人保管好，不翻开"。这是你当年对人的方式：把话摁住，不打扰、不面对。
- **正确答案（当年没做的小动作）**：把信读完，在书页空白处写一句回应（不寄出，只是不再假装它不存在）。
- **回收伏笔**：信的笔迹，慢慢和你自己的像；管理员说"这封信，等的人其实就在这馆里"。
- 物件候选：信 / 书 / 借阅台登记簿。

### 夜 B《满员的书架》—— 逃避域：家庭 / 不愿回去的屋
- **别人的书（decoy 表面）**：一个"住户"留下的整架书，逾期是因为人搬走了却不肯清。你以为是帮人整理遗留在图书馆的私人物件。
- **习惯动作（伪 HE）**：你把架子重新码齐、把空位填满，制造"一切如常"的假象。这是你当年对家的方式：用忙碌和秩序盖住缺席。
- **正确答案**：留出那个空位，不填；允许它空着。
- **回收伏笔**：架上有本你童年读过的书，书脊写着一个你熟悉到心口发紧的名字（先不点破）。
- 物件候选：整架书 / 那本童书 / 空位。

### 夜 C《雨水收集器》—— 逃避域：自我 / 咽下的情绪
- **别人的书（decoy 表面）**：有人留在窗边一个接雨水的玻璃罐，说"替我接住那些我哭不出来的日子"。你以为是帮一位陌生人保管。
- **习惯动作（伪 HE）**：你把罐子里的雨水倒掉、擦干净放回去——"情绪处理好了，没事了"。这是你当年对自己的方式：把难过清空，假装天晴。
- **正确答案**：让罐子接满，不倒；承认那些日子确实落下来了。
- **回收伏笔**：罐底沉着一枚你认得的小东西（钥匙 / 纽扣），属于"你很久以前弄丢的自己"。
- 物件候选：雨水罐 / 罐底小物 / 窗。

### 夜 D（桥）《管理员借走的书》—— 闸门显借阅人 = 玩家
- 一本被管理员"借走"、一直没还的书，系统显示借阅人栏空白。你帮他查，翻到最后一页——借阅人签名，**是你的名字**。
- 这一夜是闸门：decoy 从"别人的书"转向"你的书"。管理员第一次没有指路，而是沉默地看着你。
- **正确答案**：你第一次没有把书归架，而是拿着它，问管理员"这本书，到底是谁的"。

## 第二幕：管理员之夜（夜 C 重访 / BE 预兆 + 身份揭晓）
- 回到夜 C 的雨水收集器场景，但这次罐子满了，溢出来。管理员终于说：这些"别人的书"，每一本都是你落在馆里的自己；你是逾期最久的那一本，而他是被留下看馆的那个你。
- **揭晓纪律**：只答"他是谁"，点到即收。不说教、不审判。他只是"没逃、留下来扛的那个版本"。
- **BE 预兆**：他说，只要你还贪恋回来 / 还书 / 陪他，你就还在循环里——门其实一直没锁。

---

## 第三幕：终章《灯还亮着》—— 三拍
1. **揭晓"管理员=自己"**：整座图书馆、那盏灯，是他留下的"温柔的构造"，不是欺骗，是陪伴用心。
2. **逃避开心的夜**：相认后特写一夜，你和那个留下的自己像真正相处那样过了一夜——轻松、甚至开心，越甜越为分离蓄力。全作最暖一夜。
3. **揭晓逃避真相 + 抉择分离**：循环本身就是最后一道逃避——你贪恋"回来 / 还书 / 陪他玩"，等于不肯走出。门没锁、灯留原地。你最终亲手做当年没做的事：走出门，不回头。
- **收束句**：「灯还亮着。——我可以一直陪伴在你身边，但你得自己往前走了。」
