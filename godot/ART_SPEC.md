## 0. 总美术方向（必读，所有画面遵守）

**类型**：互动叙事 / 文字冒险，锈湖（Rusty Lake）式点击探索 + 克制悬疑。
**整体气质**：杂志摄影质感、亚洲城市夜景、便利店灯光般**克制而安静**。湿润、低饱和、暖黄点光在冷蓝里。
**视觉隐喻**（用户偏好，来自化学/实验背景）：用「湿润反光、容器接水、墨迹晕开、灯不灭」承载情绪，不要直白 symbol。

### 参考（Visual References）
- 氛围：Rusty Lake 系列（Cube Escape / The White Door）静态场景构图与颗粒感
- 光影：深夜便利店玻璃门暖黄透射、雨夜城市霓虹朦胧反射
- 质感：胶片杂志内页、轻微噪点、不过度锐利

### 色彩系统（全局统一）
| 角色 | 色值（建议） | 说明 |
|---|---|---|
| 夜色基底 | `#1a1f2b` → `#0e1118` | 冷蓝灰，场景最深背景 |
| 暖点光 | `#f4c77b` / `#e8a85a` | 馆内灯、便利店式暖黄，唯一高饱和来源 |
| 墨/文字 | `#2b2b2b` 手写 `#3a3326` | 近景墨迹、便签 |
| 雨/水 | `#6f8aa6` 半透明 | 湿润反光、盆中水 |
| 禁忌高亮 | 禁用纯红 `#ff0000`、霓虹紫、AI 发光描边 | 保持克制 |

### 禁忌（Anti）——违反任一条即打回
- ❌ AI 生成感（塑料光滑、过度发光描边、无意义复杂纹样）
- ❌ 电商/扁平插画干净感（纯色块、无颗粒、无情绪）
- ❌ 日式符号堆砌（狐面、鸟居、和风纹样等）——本作是亚洲都市夜，不是日式
- ❌ 过度饱和、强对比、戏剧化打光

### 画幅与分辨率
- **主画幅 16:9**，基准 **1920×1080**，交付 **@2x = 3840×2160** PNG。
- 移动端（TapTap）需额外 **9:16 = 1080×1920 @2x** 竖版构图变体（大场景背景需重构图，非简单裁切）。
- 近景 CG / 道具 / 角色：**透明通道 PNG**。

### 交付格式
- 背景大场景：`.psd` 分层（背景层 / 中景色块 / 前景物件 / 独立光影层）+ 导出 `.png`。
- 道具 / 角色：`.psd`（线稿/上色/光影分层）+ 透明 `.png`。

## 1. 大场景背景（8 区 × 夜氛围变体）

> 8 个真实区域（`void_room` 永不开启，不画）。每个区域给**基础构图** + **夜次氛围变体**。
> **场景层模型（重要）**：近景分两层——① `scenes.json` 里的**恒定场景层**（同一物件、同一文本，每夜都出现）；② 各夜 JSON 里的**逐夜专属层**（当夜才出现的剧情物）。画恒定层时，物件与文案必须跨夜一致；画逐夜层时按当夜剧情。

### BG-01 `entry_porch` 雨夜门廊
- **内容**：图书馆外侧门廊，雨夜。左侧一道门（木框+上半玻璃，门内透暖黄灯）；右侧墙角斜倚一把伞（伞尖垂地）。地面湿反光，映门内暖光与远处城市霓虹朦胧块。
- **构图**：门占左 1/3，伞在右前景低角，中部留"门槛"空间。
- **夜次变体**：序章/A·B·C 雨大、伞尖滴水；夜D 雨小伞尖不滴；夜E 门从内被抵着（门缝透光+门后人脸剪影）；夜F/Z 门微开没锁、光泻出更多。
- **尺寸**：1920×1080 @2x，伞/门独立图层。

### BG-02 `service_desk` 服务台 / 借阅台
- **内容**：室内长木服务台横贯下 1/3，台后书架墙（满员书架意象）。台面：登记册、笔、胶水、旧书、那张"写了又划的纸"、凉透的茶（恒定场景层 `his_tea`）、夜Z 起台面摆序章旧书。
- **构图**：服务台前景横条，玩家站台前（第一人称不画全身），台后书墙与灯。
- **夜次变体**：A·B·C 台面有划掉"回来看看吧"的纸；夜D 纸更显眼；夜E"不急"茶消失、登记册合着；夜Z 桌上写好未寄的通知+序章旧书单独摆。
- **尺寸**：1920×1080 @2x，台面物件独立图层。

### BG-03 `reading_room` 阅览区
- **内容**：安静阅览室。一排桌椅（其中"你常坐的位子"椅背朝外，恒定场景层 `turned_chair`）；靠墙旧灯常亮（恒定 `old_lamp`）；侧墙书架。
- **构图**：纵深感，桌椅引向远处旧灯；朝外椅居中可识别。
- **夜次变体**：常规旧灯亮、座位空；夜D 旧灯旁他侧影；夜F 空位过道视角；夜Z 三物证墙。
- **尺寸**：1920×1080 @2x。

### BG-04 `stacks_deep` 书库深处
- **内容**：高耸书架通道，逼仄、仅顶部一线光。最里头一面架子**漏雨**（搪瓷盆接水，旁塑料桶+两饮料瓶）。某本书里夹信/相册/《一屋子的雨》。
- **构图**：书架纵深走廊，漏雨角在尽头右侧。
- **夜次变体**：A·B·C 夹信/相册/《一屋子的雨》在架上、漏雨盆在；夜D 盆在、墙上便条"别怕，是雨自己要下的"；夜E 盆边"雨天物"收进小盒。
- **尺寸**：1920×1080 @2x，漏雨角独立图层。

### BG-05 `study_zone` 自习区（留灯仪式落点）
- **内容**：一排自习灯，其中**仅一盏长亮、对应座位空着**（恒定场景层 `keep_lamp`，"替你留的灯"）。
- **构图**：灯排横列，亮的那盏偏中央，空位在灯下，留白多。
- **夜次变体**：A·B·C·D 灯亮位空；夜G·H·I 他走到灯边把灯"往空位偏"（动效）；夜Z 灯还亮着。
- **尺寸**：1920×1080 @2x，单盏亮灯独立图层。

### BG-06 `utility_zone` 便民配套区
- **内容**：失物招领信箱（投递槽，逐夜任务载体 `lost_found`）、储物柜列、打印区、已还书目墙（跨夜累积状态）。
- **构图**：信箱居中前景，书目墙在侧，储物柜/打印背景填充。
- **夜次变体**：A·B·C 信箱满（投递槽张着口）；夜D 起/夜Z 信箱空了（"你那夜还回去了"）；夜Z 登记册翻到你那页。
- **尺寸**：1920×1080 @2x，信箱/书目墙独立图层。

### BG-07 `lounge_stairs` 休息室 / 楼梯（**管理员专属，序章/A·B·C 锁定**）
- **内容**：管理员休息室，木桌、椅、墙角凉茶、墙上便条。桌子压一本"逾期最久"旧书。
- **解锁**：夜D 起才开放探索（前期门锁着，所以早期夜晚看不到——这是有意闸门，非缺失）。
- **夜次变体**：夜D 桌上旧书+"逾期最久"登记、抽屉旧信露角、凉茶凉了；夜E 桌空着（书收起）。
- **尺寸**：1920×1080 @2x。

### BG-08 `archive_lamp` 灯控室（**管理员专属，序章/A·B·C 锁定**）
- **内容**：整面墙灯开关（主灯开关在其中），铁柜插满逾期借阅卡，中央一盏**主灯**（罩子发黄、光稳）。
- **解锁**：夜D 起开放；夜G/H/Z 各带专属近景。
- **夜次变体**：夜D 铁柜开、借阅卡墙（三 decoy 卡签名=你）；夜G 主灯特写；夜H 半开档案室门；夜Z 回声/灯里小人。
- **尺寸**：1920×1080 @2x，开关墙/铁柜/主灯独立图层。

## 10. 交付与命名规范

```
godot/art/
├── bg/            # 大场景背景（BG-01~08，变体双下划线）
├── closeup/       # 近景 CG（按 §2 热点 ID 命名，如 cu_entry_porch__umbrella.png）
├── prop/          # 道具（P-xx）
├── char/          # 角色（CH-xx / HD-xx）
├── ui/            # UI 组件（UI-xx）
├── fx/            # 动效序列帧/粒子
├── audio/         # 音轨（AU-xx）
└── shell/         # 系统外壳（SH-xx）
```
命名：`{类别前缀}-{ID}_{描述}.png`，变体 `{前缀}-{ID}_{描述}__{变体名}.png`。
所有透明图保留 alpha；背景保留可重构图层。
