## 0. 总美术方向（必读，所有画面遵守）

**类型**：互动叙事 / 文字冒险，锈湖（Rusty Lake）式点击探索 + 克制悬疑。
**整体气质**：杂志摄影质感、亚洲城市夜景、便利店灯光般**克制而安静**。湿润、低饱和、暖黄点光在冷蓝里。
**视觉隐喻**（用户偏好，来自化学/实验背景）：用「湿润反光、容器接水、墨迹晕开、灯不灭」承载情绪，不要直白 symbol。

### 参考（Visual References）
- 氛围：Rusty Lake 系列（Cube Escape / The White Door）的静态场景构图与颗粒感
- 光影：深夜便利店玻璃门的暖黄透射、雨夜城市霓虹的朦胧反射
- 质感：胶片杂志内页、轻微噪点、不过度锐利

### 色彩系统（全局统一）
| 角色 | 色值（建议） | 说明 |
|---|---|---|
| 夜色基底 | `#1a1f2b` → `#0e1118` | 冷蓝灰，场景最深背景 |
| 暖点光 | `#f4c77b` / `#e8a85a` | 馆内灯、便利店式暖黄，唯一高饱和来源 |
| 墨/文字 | `#2b2b2b` 手写 `#3a3326` | 近景墨迹、便签 |
| 雨/水 | `#6f8aa6` 半透明 | 湿润反光、盆中水 |
| 禁忌高亮 | 禁用纯红 `#ff0000`、霓虹紫、AI 发光描边 | 保持克制 |

### 禁忌（Anti）——违反任一条即打回
- ❌ AI 生成感（塑料光滑、过度发光描边、无意义复杂纹样）
- ❌ 电商/扁平插画干净感（纯色块、无颗粒、无情绪）
- ❌ 日式符号堆砌（狐面、鸟居、和风纹样等）——本作是亚洲都市夜，不是日式
- ❌ 过度饱和、强对比、戏剧化打光

### 画幅与分辨率
- **主画幅 16:9**，基准 **1920×1080**，交付 **@2x = 3840×2160** PNG。
- 移动端（TapTap）需额外 **9:16 = 1080×1920 @2x** 竖版构图变体（大场景背景需重构图，非简单裁切）。
- 近景 CG / 道具 / 角色：**透明通道 PNG**，尺寸按下面各条标注。

### 交付格式
- 背景大场景：`.psd` 分层（背景层 / 中景色块 / 前景物件 / 独立光影层）+ 导出 `.png`（含透明可选）。
- 道具 / 角色：`.psd`（线稿/上色/光影分层）+ 透明 `.png`。
- 命名见 §10。

## 1. 大场景背景（8 区 × 夜氛围变体）

> 8 个真实区域（void_room 永不开启，不画）。每个区域给**基础构图** + **夜次氛围变体**（变体靠灯光/物件状态区分，可同一底图换光层）。

### BG-01 `entry_porch` 雨夜门廊
- **内容**：图书馆外侧门廊，雨夜。画面偏中景，玩家视角站在门槛。左侧一道门（木框+上半玻璃），门内透出暖黄灯光；右侧墙角斜倚一把伞（伞尖垂地）。地面湿反光，映出门内暖光与远处城市霓虹朦胧块。
- **构图**：门占左 1/3，伞在右前景低角，留中部"门槛"空间给玩家站位与雨。
- **色调**：冷蓝夜 + 门内暖黄点光，地面高反光，颗粒感。
- **夜次变体**：
  - 序章 / 夜A·B·C：雨大，伞尖明显滴水（加雨粒子 + 地面水痕）。
  - 夜D：雨小，伞尖**不再滴水**（文案"雨小了"），光感稍静。
  - 夜E：门**从内被抵着**，台灯光打在门后人脸剪影（人脸用 §4 角色近景，此处仅门缝透光）。
  - 夜F / 夜Z：门**微开/没锁**，可自己推开，光从门内泻出更多。
- **尺寸**：1920×1080 @2x，伞/门做独立图层便于状态切换。

### BG-02 `service_desk` 服务台 / 借阅台
- **内容**：室内。一道长木服务台横贯下 1/3，台后书架墙（满员书架意象，书脊密排）。台面有：登记册、笔、胶水、一摞旧书、那张"写了又划的纸"、夜F 起台面摆三物证。顶上一盏主灯（暖黄）。
- **构图**：服务台为前景横条，玩家站台前（第一人称，不画全身），台后空间放书墙与灯。
- **色调**：暖黄主灯为唯一高光，四周压暗，木质暖棕。
- **夜次变体**：
  - 夜A·B·C：台面有那张划掉"回来看看吧"的纸、抽屉里空白便签。
  - 夜D：台面纸更显眼（"你的书逾期了"清晰，划痕可见）。
  - 夜E：桌上"不急"茶**消失**、登记册合着（空位感）。
  - 夜F：台面**三物证并排**（便签/相册书/雨天物盒）。
  - 夜G·H·I：他主动亮灯/炫耀，整体暖光更满、更"甜"。
  - 夜Z：桌上**写好未寄的通知** + 角落序章旧书单独摆。
- **尺寸**：1920×1080 @2x，台面物件独立图层。

### BG-03 `reading_room` 阅览区
- **内容**：安静阅览室。一排读者桌椅（其中"你常坐的位子"椅背搭一块除尘布），靠墙一盏旧灯常亮，侧墙书架。夜Z 起一面墙摆三物证。
- **构图**：纵深感，桌椅成排引向远处旧灯；除尘布座位居中可识别。
- **色调**：比门廊更"内收"的暖，旧灯柔光，书墙冷调压暗。
- **夜次变体**：
  - 常规：旧灯亮，座位空着（除尘布在）。
  - 夜D：旧灯旁他"说漏嘴"（仅文案，画面加他侧影近景）。
  - 夜F：空位过道视角（玩家站过道口看空位）。
  - 夜Z：阅览区**三物证墙**（便签/相册书/雨水盆边物并排上墙）。
- **尺寸**：1920×1080 @2x。

### BG-04 `stacks_deep` 书库深处
- **内容**：高耸书架通道，逼仄、只顶部一线光。最里头一面架子**漏雨**（地上搪瓷盆接水，旁摞塑料桶+两饮料瓶）。某本书里夹信/相册/《一屋子的雨》。
- **构图**：书架形成纵深走廊，漏雨角在尽头右侧，光从尽头上方可视。
- **色调**：最暗场景，仅顶部冷光 + 漏雨角微反光；雨声视觉化（水痕、盆中涟漪）。
- **夜次变体**：
  - 夜A·B·C：夹信/夹相册/《一屋子的雨》在架上；漏雨盆在。
  - 夜D：盆还在，墙上便条"别怕，是雨自己要下的"（手写体道具）。
  - 夜E：盆边"雨天物"被收进小盒（标"雨天物"）。
- **尺寸**：1920×1080 @2x，漏雨角独立图层（盆/桶/瓶/水）。

### BG-05 `study_zone` 自习区（留灯仪式落点）
- **内容**：一排自习灯，其中**仅一盏长亮、对应座位空着**（"替你留的灯"）。氛围最"温柔的构造"。
- **构图**：灯排横列，亮的那盏偏画面中央，空位在灯下，留白多。
- **色调**：全作最暖的一束光，四周暗，克制安静。
- **夜次变体**：
  - 夜A·B·C·D：灯亮、位空，他偶尔站灯边。
  - 夜G·H·I：他走到灯边把灯"往空位偏"（动效 §6），暖意加重。
  - 夜Z：灯还亮着，他记最后一次你坐这的样子。
- **尺寸**：1920×1080 @2x，单盏亮灯独立图层（便于"偏移"动效）。

### BG-06 `utility_zone` 便民配套区
- **内容**：失物招领信箱（投递槽）、储物柜列、打印区、已还书目墙（跨夜累积状态）。信箱是核心交互物件。
- **构图**：信箱居中前景，书目墙在侧，储物柜/打印作背景填充。
- **色调**：公共空间冷白光 + 信箱暖旧木色。
- **夜次变体**：
  - 夜A·B·C：信箱**满**（投递槽张着口，内部积物，见 §2 CU-A3）。
  - 夜D 起 / 夜E / 夜Z：信箱**空了**（"你那夜还回去了"）。
  - 夜Z：登记册翻到你那页（"压得最久"）。
- **尺寸**：1920×1080 @2x，信箱/书目墙独立图层。

### BG-07 `lounge_stairs` 休息室 / 楼梯（管理员专属，夜D 起解锁）
- **内容**：管理员休息室，木桌、椅、墙角一杯凉茶推在"不急"位、墙上便条。桌子压一本"逾期最久"的旧书。
- **构图**：私密小空间，桌居中，便条在墙，茶在桌角。
- **色调**：比服务台更私密暖，像"他的家"。
- **夜次变体**：
  - 夜D：桌上旧书+"逾期最久"登记、抽屉旧信露一角、凉茶凉了。
  - 夜E：桌**空着**（书收起，"提前收拾自己东西"）。
- **尺寸**：1920×1080 @2x，桌面物件独立图层。

### BG-08 `archive_lamp` 灯控室（管理员专属，夜D 起解锁）
- **内容**：整面墙的灯开关（主灯开关在其中），铁柜插满逾期借阅卡，中央一盏**主灯**（罩子发黄、光稳）。
- **构图**：开关墙占主导，铁柜在侧，主灯居中偏下暖光。
- **色调**：机械冷感墙 + 主灯唯一暖核，对比强烈但克制。
- **夜次变体**：
  - 夜D：铁柜开，借阅卡墙（三物证签名=你，更早三张涂墨"下次"）。
  - 夜G：主灯特写（"是我装的"，见 §2 CU-G1）。
  - 夜H：半开的档案室门（序章旧书"等你还"，见 §2 CU-H1）。
  - 夜Z：灯控室回声（他=你当年腔调，见 §2 CU-Z2）。
- **尺寸**：1920×1080 @2x，开关墙/铁柜/主灯独立图层。

```
godot/art/

│   ├── bg_entry_porch__door_open.png      # 变体用双下划线

│   ├── bg_stacks_deep.png
│   ├── bg_study_zone.png
│   ├── bg_utility_zone.png
│   ├── bg_lounge_stairs.png
│   └── bg_archive_lamp.png
├── closeup/       # 小近景 CG（CU-xx）
├── prop/          # 道具（P-xx）
├── char/          # 角色（CH-xx / HD-xx）
├── ui/            # UI 组件（UI-xx）
├── fx/            # 动效序列帧/粒子
├── audio/         # 音轨（AU-xx）
└── shell/         # 系统外壳（SH-xx）
```
命名：`{类别前缀}-{ID}_{描述}.png`，变体 `{前缀}-{ID}_{描述}__{变体名}.png`。
所有透明图保留 alpha；背景保留可重构图层。
