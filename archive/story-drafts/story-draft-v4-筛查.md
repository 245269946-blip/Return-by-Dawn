# 《逾期之书》v4 第一性原理 + 对抗性筛查

> 目的：把 v4（纯净确认版）对照"我们的预期"（不可动摇的设计原则）过一遍，回答"是否符合预期"。
> 结论先行：**v4 在结构上已达成预期**，且在蒸馏中修掉了 v3 埋着的两个真实缺陷（夜 D 点破、借阅卡夜 D/夜 H 冲突）。剩余风险全部是"prose 执行层的 guard 要求"，不是设计层矛盾——已逐条给出写作者红线。

## 二、第一性原理筛查（剥离叙事包装，回到本质）

> 方法：把每个核心支柱的"技巧外壳"剥掉，问——**即使没有花哨机制，这个设计的本质是否成立、是否服务于最高指令？**

### 2.1 脊柱（P1）：「书=遗忘碎片，馆员=留下的自己」
- **本质**：把"逃避的复利"具象成一个人格分裂式结构——你逃一次，掉一个版本的自己；留下的那个你成了管理员，一直等你回来。
- **成立性**：✅ 成立。它让抽象的"自我断裂"变成可感知的空间（图书馆）与可互动的人（馆员），正是锈湖式节点叙事的骨架。莫比乌斯环成立的前提只有一条：**前期绝不泄底**（见 P8 筛查）。
- **结论**：本质牢。

### 2.2 逃避引擎（P5）：「当下有效→复利破碎」
- **本质**：每次逃避是"用负责的样子绕开情感核心"，当下止痛（伪 HE），长期遗落自己。
- **成立性**：✅ 成立。序章/夜 A/B/C 均呈现"做完→轻松→未了结"，伪 HE 在每个夜都兑现；夜 F 起"轻松变薄、觉悟变厚"是引擎的二阶表现。
- **注意点**：引擎要成立，**努力动作必须真存在**（v4 §二已钉死）。若 prose 把逃避写成"懒/坏"，引擎塌。
- **结论**：本质牢。

### 2.3 BE 温度（P4/P6）：「限度温柔，非致郁」
- **本质**：不靠惨剧炸场，靠"所有人都尽力了、但只能到这里"的认知落点制造重量。
- **成立性**：✅ 成立。终章收束句"灯还亮着…你得自己往前走"、callback"懂了也还是难"、双结局都落到"只能到这里"——全部对齐 P4/P6。
- **风险（第一性）**：乙结局"往前走"在 stripped 视角下，最容易被读成"痊愈/净化"（违背 P5 弧光对照非净化）。v4 已在乙衔接句加 guard（"带着知道却曾做不到的自己"），但 prose 仍易滑。→ 列为执行红线。
- **结论**：本质牢，但乙落点需 guard。

### 2.4 decoy（P2）：「前期别人的事，后期是我」
- **本质**：先让玩家当"解谜者/助人者"，再回收成"被解的那个"。
- **成立性**：✅ 夜 A/B 显式"当别人落的/别的读者遗落物"，镜面映射只到"一沉"；夜 C 靠漏雨任务+漏雨旧事书承载。
- **风险**：夜 C 的 decoy 弱于 A/B（C 是"自己闪回"多于"别人故事"，虽有漏雨旧事书补，但 prose 重点在任务）。→ 见对抗性 R3。
- **结论**：本质牢，C 需补强。

### 2.5 自认（P8）：「情绪 uncanny，非推理；夜 D 合拢」
- **本质**：让玩家"慢慢不对劲、最后被戳穿"，不是"逻辑推出来是我"。
- **成立性**：✅ §九 四条常量线索（墨团/镜面/常量/管理员暗线）是情绪铺垫，夜 D 借阅卡+种子合拢是 click 时刻，符合"诡异熟悉感"。
- **结论**：本质牢，但执行依赖写作者不点破（见对抗性 R1）。

## 三、对抗性筛查（主动攻击，找裂缝 / 矛盾 / 泄底）

> 方法：站在"最坏写手 / 最挑剔玩家"视角，逐条攻击 v4，标记严重度。
> **【已修】** = v4 相对 v3 已解决的；**【风险】** = 需 prose 阶段 guard；**【建议】** = 可选增强。

### R1【已修·严重】夜 D 原"你不就是你自己吗"软点破，违反 P8 与 P2
- **攻击**：v3 夜 D 直接写"这不就是你自己吗？"——这是作者/叙述者替玩家点破，既踩 P8 红线（A/B/C 之外也不该由旁白宣布），又提前泄 decoy。
- **v4 修正**：改为"你听着，心里一种说不清的熟悉感慢慢浮上来" + 借阅卡签名=你 + 三夜种子回照 → 玩家自己"啊原来一直是我"。旁白不宣布，只给 click 物料。
- **判定**：✅ 已解决，且与 §9.2 闸门纪律自洽。

### R2【已修·严重】借阅卡"签名=你"在夜 D 与夜 H 两处冲突（v3 真矛盾）
- **攻击**：v3 中 §8.2 把借阅卡定为夜 D 闸门机关，但夜 H（第三幕）又写"某一夜他不小心让你瞥见借阅卡签名=你…你没深想，只当怪规矩"并标"这就是夜 Z 伏笔"。而夜 D（第二幕）早于夜 H（第三幕）——逻辑上不可能"夜 H 第一次给线索"又"夜 D 已合拢身份"。这是 v3 的真实时间线矛盾。
- **v4 修正**：借阅卡唯一归 **夜 D 闸门**（P11 同步改为"借阅卡=夜 D 机关"）；夜 H 改为种"序章续借本·等你还"的夜 Z 种子（玩家此时已知"是我"，仍贪恋甜、不碰"该还"），不再重复揭示身份。铺垫回收清单同步改：原"通知单笔迹→夜 D/夜 H"改为→夜 D；新增"序章续借本等你还→夜 H 种/夜 Z 收"。
- **判定**：✅ 已解决。夜 D 早揭示反而强化第三幕"明知却贪恋"的逃避层次。

### R3【风险·中等】夜 C 的 decoy 弱于 A/B
- **攻击**：夜 A/B 玩家明确"当别人的遗物"，镜面清晰；夜 C 玩家直接做漏雨任务+自己闪回，"别人的书"框架弱（仅"漏雨旧事书"一笔带过）。若 prose 把 C 写成"我自己的回忆录"，decoy 在 C 段提前破。
- **缓解（v4 已做）**：夜 C 明确加入"漏雨旧事书（别人的遗落物），借阅人栏同样涂烂"，与 A/B 同构。
- **写作者红线**：漏雨旧事书必须作为"别人故事"载体出现，镜面映射（"用一件件的事填满"）走"旁观别人"角度；玩家自己的闪回是"回声触发"，不是"这书就是我"。
- **判定**：⚠️ 设计已补，执行需 guard。

### R4【风险·中等】夜 A 管理员"很多人把想说的话先写在别的地方"边界
- **攻击**：这句算不算轻微说教/暗示，踩 P7（零说教）？
- **分析**：它是功能互动（递胶水便签）附带的随口观察，无评价、无劝导，且玩家"没接"——属 P7 允许的"暗示"非"说教"。但若 prose 加重语气（"你应该…"）即越界。
- **写作者红线**：保持随口、无评价、玩家可忽略；管理员任何台词不得出现"你该面对/你逃避了"类。
- **判定**：⚠️ 边界 OK，执行需 guard。

### R5【风险·中等】乙结局"往前走"滑向救赎/净化（违背 P5）
- **攻击**：乙落点"门没锁走出""肯往前走"在 stripped 视角极像"痊愈"，与 P5"弧光是对照非净化、不是你痊愈了"冲突。
- **v4 已做 guard**：乙衔接句明写"不是我痊愈了，是带着知道却曾做不到的自己肯往前走"；callback 核心句"懂了也还是难"把限度从无知改成生而为人的难。
- **写作者红线**：终章任何位置不得出现"他/她终于好了/解脱了/走出了阴影"；"往前走"必须伴随"带着未完成/带着迟了的遗憾"。
- **判定**：⚠️ 设计已 guard，执行需 guard。

### R6【风险·低】夜 B 体量与 ~15min 节奏 / decoy 轻感
- **攻击**：夜 B 动机块极长（童年吵架+时代脱节+互相回避+具体疏忽+不回家逻辑链），若全量塞进单夜 prose，可能破坏"轻日常→重揭示"的落差节奏，且 INFO-DUMP 会提前泄情绪 payload。
- **建议**：动机块作为可展开"深挖节点"——表层仍走"翻相册→闪回一沉→发红包"流程，深层动机在玩家停驻/回看时释放（契合锈湖式节点探索）。避免一口气灌输。
- **判定**：⚠️ 可选结构优化，不矛盾。

### R7【风险·低】"玩家是常量"③ 与"管理员暗线"④ 尚未进 prose
- **攻击**：v4 把③伞/红包/做完呼应、④管理员记得细节，只写在每夜"自认播种"块，未进夜 A/B/C 正文。若 prose 不 weave，这两类线索埋不住，夜 D 合拢会显得空降。
- **写作者红线**：③ 极轻植入（伞/红包/做完=随手动作，玩家可忽略）；④ 通过"跨夜伏笔被收着/单独摆/盆边物在"的**玩家发现**呈现，而非管理员说破。
- **判定**：⚠️ 执行需落地（已知"道具落地"下一步）。

### R8【一致性钩稽·通过】关键物证跨夜链条
- 序章"桌上压着一摞没还的书" ↔ 夜 D"桌上压着一本逾期最久的书"：✅ 一致（种子→回收）。
- 夜 A/B/C 跨夜伏笔①②③（便签/相册书/雨水盆）↔ 夜 F 三物证合流：✅ 一致。
- 序章续借本 ↔ 夜 H"等你还" ↔ 夜 Z"带书回来"：✅ 一致（v4 已理顺）。
- 通知单笔迹越来越像自己 ↔ 夜 D 借阅卡签名=你：✅ 一致。
- **判定**：✅ 全链闭合，无钩稽断裂。

### R9【信息·低】区域≠线索一一对应（P9）仅原则声明，未落夜 spec
- **攻击**：v4 §一 P9 写了原则，但各夜 specs 未显式标注"区域隐喻 / 网状散布"。这与"道具落地"待办一致，不矛盾，但易在 prose 时退化成"每区一条线索"。
- **建议**：道具落地阶段，为每夜标注"线索安放区域 + 该区域展示隐喻"，确保网状而非线性。
- **判定**：⚠️ 已知下一步，不阻塞。
