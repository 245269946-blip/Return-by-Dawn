## 一、不可动摇的设计原则（北极星与纪律）

1. **脊柱（不可换）**：图书管理员 = 逾期最久的那本书 = 玩家自己遗落的版本。莫比乌斯环：书是遗忘的碎片，馆员是留下的自己。此真相属后期揭晓（夜 D 闸门 → 终章），前期只立馆员人设与"还书循环"，绝不提前说破。
2. **表层 decoy**：前期每一夜让玩家以为在"帮别人还书 / 解送错通知的谜"，先不引到自己身上；第二、三幕回收成"其实是你"。decoy 命门 = 玩家前期必须读成"别人的事"。
3. **北极星**：To the Moon 式情绪叙事优先。无惩罚式失败——每夜都通向一个结果（归档 / 归还 / 私藏 / 隐秘），非通关 / 失败。
4. **BE 温度（最高指令）**：落点非纯惨纯痛，而是"真实、温馨、所有人竭尽全力做到无憾、但只能到这里"。限度温柔接受，不滑向为悲而悲。任何痛都必须落在这句话上。
5. **逃避引擎（机制核心）**：每次逃避"当下有效"（即时止痛 → 成瘾 → 循环成立），但把自己"对（倒）下" → 遗落一个版本的自己 → 复利累积成破碎。玩家重复做出"当年逃避的习惯动作"，当下利好 = 伪 HE；累积到揭晓管理员身份与真相时才生反差。这不是正向改变 / 救赎故事——弧光是"对照"而非"净化"。变成"你痊愈了"会软化"只能到这里"。
6. **认知冲击 ≠ 事件冲击**：反对加事件炸弹（惨剧 / 死亡 / 背叛，会击穿 BE 温度 + 致郁否定）。靠① 落差（前期越轻日常 → 揭示越重）② 渐进拼合（多线索 → 玩家亲口说"是我"）③ 限度温柔落点（"只能到这里"因所有人都尽力了反而最重）。校准样本《海边的曼彻斯特》。
7. **管理员行为边界（写死）**：只陪伴 + 寄逾期通知，零说教 / 零强提示 / 零道德高位。把你唤回的正是他寄的通知（出于爱、也成锁链）。终章分离 = 他停寄通知 / 你不再应答。管理员非道德优胜者——他与玩家是同一个人，都只是累了；"好"是陪伴用心非道德高位。
8. **自认红线**：A/B/C 绝不点破"其实是你"，否则 decoy 命门塌、莫比乌斯环失效。自认只发生在夜 D 闸门，且由玩家自己合拢（拼图合拢，不是被告知）。
9. **区域 ≠ 线索一一对应**：区域只是线索安放的地方，须与图书馆场景适配、带对应展示隐喻；故事由多线索跨区互动触发，网状散布，非沿区域线性推进。
10. **双结局权重**：甲（沉溺）/ 乙（告别）并列成立，无主从，UI 不作"错选"暗示。
11. **"续借"呈现口径**：序章只教动作、不解释意义。借阅卡「签名 = 你、借出日 = 序章续借那天」作为**夜 D 自认闸门**的核心机关出现（延后于序章、不剧透）；夜 H 另种「序章续借本一直被留着、等你还」的夜 Z 种子，不重复揭示身份。
12. **终章 callback 落点**：第一幕夜 A/B/C"正确做法"闪回落在「知 ≠ 行」缝隙，不写成全做到（防软化 BE）；用夜 F 三物证触发。夜 B 专属"理解父母 → 放下傲慢 → 和解"留后续，callback 只到"知"的承认。

## 二、核心循环（每夜结构，内心状态在变）

```
馆内处境（回声触发：与"已逃避过"高度相似的处境，图书馆重发通知单/指派任务）
   ↓
努力动作（确认：玩家 genuine 挣扎 / 冲动想做对的——摸到笔、翻到号码、鼻子发酸）
   ↓
习惯逃避 = 走捷径的完成方式（把事"做完"了，但绕开真正难的那一下 → 当下止痛 = 伪HE）
   ↓
遗落一个版本的自己（碎片）
   ↓
管理员功能互动 + 空间暗示（馆内物证跨夜累积）
```
- 逃避 ≠ 做错事。是"用看起来负责的方式，绕开了情感核心"。
- 努力动作必须存在。没有挣扎就直接逃避，玩家会觉得自己"本来就是烂人"，失去对照意义。
- 伪 HE = 走捷径完成后的即时轻松。前期真轻松，后期轻松变薄；二、三幕靠"轻松变薄、觉悟变厚"驱动。

## 三、序章·续借（教学关）

- **馆内处境**：你收到一张逾期通知单（你以为是新发的，其实是图书馆按老习惯重发的）。手里有一本迟迟没还的书。管理员温和提醒：「可以续借，或者——下次来再还也行。」
- **努力动作**：你今天其实有空，可以现在去还。你犹豫了一下，书都掏出来半本。
- **习惯逃避（走捷径完成）**：你说「下次吧」，把书塞回包里，办了续借。通知单「处理掉了」，书还在你这儿。逃避的最小单元：用「延后 / 续借」替代「真正了结」。
- **管理员功能互动**：他熟练地帮你办续借，动作温柔。你瞥见他桌上也压着一摞没还的书（种子：他也是逾期者 / 他 = 你）。
- **遗落碎片**：无（教学），但种下「管理员的温柔 = 纵容」的种子——后期才懂，他每一次「不急」「下次吧」，都是在替你留后路，也替自己留你。
- **设计意图**：体验一次最小循环（通知单 → 回声 → 续借 → 轻松 → 未了结），且立刻确立「管理员强引导继续逃避」是图书馆的底层温柔。

## 四、第一幕·碎片（三域，表层 decoy 沉浸期，全在图书馆内）

> decoy 保留：每夜你先以为处理的是"别人的遗物 / 别人的事"，后期回照成"是你自己的碎片"。每夜的处境、努力、逃避、伏笔全部在图书馆业务里发生。下文每夜附「自认播种」四项，对应第九节机制。

### 夜 A《夹在书里的信》— 关系域（没说出口的话）

- **馆内处境（回声）**：你还书时，一本书里掉出一封没署名的信。管理员让你处理：「遗失物品，交服务台，或者物归原主得找到本人。」你下意识当它是别的读者落的。
- **偶然的重 + 闪回**：你翻开信，写的是对某个人没说出口的、很重的话——与你自己的某段关系回声。看到某一句时，你突然记忆闪回：当年你也有这样一封信 / 这样一个人，你当时也是"多一事不如少一事"，把话咽了回去，假装没看见。闪回不揭底，只让你自己心里一沉——"我当初就是这么逃的"。
- **努力动作（确认）**：闪回过后，你冲动地想：也许我该对生命里那个人，也写点真的。你摸到笔，在便签上写了两句真话，手在抖。
- **习惯逃避（走捷径完成）**：你把便签揉了，把信按规矩贴标签放进失物招领信箱（任务"完成"了）。你秉持「多一事不如少一事」——不是接着去解决，而是送往信箱了事。你"做了该做的"，当下轻松。但真话没寄给该寄的人，也没说出口。关系域的逃避 = 用"正确流程"替代"真正表达"。
- **正确答案（当年没做）**：当年你该对那个人说出 / 写出那句真的重话，而不是让关系停在表面和平。
- **管理员功能互动 + 暗示**：他递你胶水和便签（功能互动：帮你贴失物标签），随口说「很多人把想说的话，先写在别的地方」。你没接。后来你在服务台抽屉发现你揉掉的便签被收着了（跨夜伏笔①）。
- **遗落碎片**：一张没寄出的真话便签 → 遗落「表达版自己」。
- **自认播种（夜 A）**：① 借阅人墨团——这封信所属的那本书，借阅人栏被涂成一团墨、裁掉一角，墨团边缘隐约有手写"下次"；② 镜面映射——信里人"多一事不如少一事把话咽了"严丝合缝对应你的关系域逃避（但叙述角度是"别人的故事"）；③ 玩家是常量——你刚把伞忘在门廊，下一本"别人"的书里就有人在门廊留伞（呼应极轻，可忽略）；④ 管理员暗线——他记得你揉掉的便签被收着（伏笔①），前期读成职业习惯。

### 夜 B《满员的书架》— 家庭域（被疏忽的关系）

> **家庭内部关系的疏忽——底层动机（本夜核心，写深刻）**：
> - "让人省心的孩子"是察言观色练出来的，不是天生的。你是家里"最让人放心"的那个：读书出去、留在大城市、从不惹事、报喜不报忧。但你不是生来就这样——小时候的家，情绪是会突然变天的：父母也会因为钱、因为累、因为各自没处安放的委屈吵起来，那股火有时会莫名落到你头上。你很小就学会了看脸色，把自己的需要、怕、委屈都收起来。不是家里不爱你，是那时的家承接不住一个孩子的情绪——他们自己都在被生活推着走。于是你心里落下一句话：真实的情绪，在家里是"不安全"的。报喜不报忧，就是从这儿长出来的自我保护。
> - 他们的关心是真的，只是和这个时代脱节了。别把他们想成拿爱要挟你的人——他们是真心的，只是那份真心裹在一套旧地图里：催你考公、劝你找个稳定工作、总担心你一个人在外面吃不好睡不好、一回家就催婚催生。这些话你听着烦、觉得被冒犯，可在他们那代人的理解里，那就是"最好的爱"。他们不是不在乎你心里怎么想，是他们真的不懂——这个信息爆炸、变得太快的时代，孩子心里在经历什么，没人教过他们；他们也是带着过时的知识、第一次做父母、第一次走到今天。错位，不等于恶意。（这一层，此刻的你还看不太清、也不太愿意看——你只觉得被冒犯。真正读懂它、放下这份年轻人的傲慢，是后来的事。）
> - 于是形成了"互相回避的安静"——两边都在小心保护对方收到的那个版本。家里有事瞒着你（"别让他分心、别让他操心"）；你有事也不说（"说了他们也帮不上、只会用他们那套来劝，还让他们睡不着"）。两边都以为自己在"为对方好"，结果谁也没真正靠近谁。这不是谁对谁错，是关系上的长期疏忽——把彼此推到"报平安"的安全区，谁也不去碰那层需要被承接、却谁都不太会承接的真实。而这一夜真正被聚焦的，是你自己：是你，为了图这份"平稳"，主动选择了不再靠近。
> - 具体的疏忽事件（让逃避有重量）：有一年父亲生过一场病，家里人瞒着你没说。你后来偶然知道，第一反应不是买票回去，而是多打了笔钱、在家族群发"注意身体、别省着"。你本可以回去看看，但你没有。你不是忙，你是已经不太会用"人"去靠近家人了——只会用"钱"和"报喜"去替代"在场"。
> - 为什么不回家（"忙"只是挡箭牌，完整逻辑链）：回老家对你早已不是轻松的事。回去要面对饭桌上那些错位的关心；你想说一句"我其实挺累、那年爸生病我后怕"，可话一出口，大概率换来的是他们那套旧地图的规劝，于是你又咽了回去。你和他们之间，仿佛隔着一整个时代——明明最亲，却无话可说。再加上这些年用钱替代陪伴攒下的亏欠，和父母真的老了。"忙"只是借口。真正让你不回去的，是回去就要面对这段被你疏忽的关系，就要重新去接那份你早已不知怎么接的、错位的亲密。所以转账、红包、报平安，是你给得出来、代价又最小的那点"连接"。

- **馆内处境（回声）**：一片"家庭相册书目"区要归架。你翻到一本旧相册书，是你小时候的家——某页是年夜饭的桌，桌边空着一个你常坐的位置。你下意识把它当别的人家 / 别的读者的遗落物。
- **努力动作（确认）**：手指停在那页。你其实想过，今年该回去一趟，或至少打个长的电话、说句"我想你们了 / 那年爸生病，我其实很后悔没回来"。你拿起手机，翻到那个存了很久却很少拨的号码，指腹悬在拨通键上——可你脑子里已经预演了一遍：接通后，那句"我想你们"大概率接不住，会变成"吃饭了没 / 工作忙不忙 / 别乱花钱"，然后你们又安然滑回各自的剧本。你知道，你怕的不是拨号，是拨通后依然无话可说。
- **习惯逃避（走捷径完成）**：你没拨。只发了个红包，附一句「最近的都好，勿念」就锁屏，书归架了，整理"完成"了。你用「尽了责」的完成感，替代了真正的回去、真正的对话。家庭域的逃避 = 用"尽责 / 红包"替代"在场"，用"报喜"替代"说真话"。（这正和夜 C 是同一条根——在家庭域，逃避长成"疏忽关系、用钱替代陪伴"；在自我域，逃避长成"用做完盖情绪"。同一种本能，两个面：都是不让那个真实、脆弱、需要被承接的自己，被看见。）
- **正确答案（当年没做）**：当年你该真的回去，或真说「我想你们 / 那年我没回来，我后悔」，而不是用钱和报喜替代在场、用省事替代面对。
- **管理员功能互动 + 暗示**：他递你书立和除尘布（功能互动），你说"马上好"却很快做完。后来你发现那本相册书被他单独摆在"待你再来看"的架位（跨夜伏笔②）——他记得你翻得最久的那页，也记得那个空着的位置。
- **遗落碎片**：一个没拨出的号码 / 一句没说的"我想你们" → 遗落「归家版自己」。
- **自认播种（夜 B）**：① 借阅人墨团——相册书的借阅人栏同样涂烂、边缘"下次"；② 镜面映射——相册书里"年夜饭空位 + 只发红包"对应你的家庭域逃避；③ 玩家是常量——你刚发完红包，书里那家人也只发红包；④ 管理员暗线——他记得你翻最久的空位页（伏笔②）。

### 夜 C《雨水收集器》— 自我域（咽下的情绪）

> **"用忙碌麻痹难过"是怎么来的（前置记忆，本夜闪回讲清）——同一时期的多重打击**：
> - 那段日子（你后来不愿细想的某一年）：一边是家里长辈离世，你没能赶回去见最后一面（正扣夜 B：那阵"忙"也把你从家拽走了）；一边是工作上遇到一个很烂的领导，天天施压、甩锅、让你怀疑自己。两件事叠在同一时期砸下来。
> - 你当时怎么做：你逞强——不是摆烂，是更用力地"把事做完"。长辈的后事、领导的交付，你一件件扛下来，做完一件划掉一件，心里暗暗相信「只要把这些都做完了，我就好了」。你确实把"事"都解决了：葬礼办妥、项目交付。但你没发现自己只解决了事，没解决自己——那股被压进去的疼，你一天也没碰过。
> - 机制落成：你发现"只要手不停、把事做完，心就不那么疼"。从那以后，任何难过袭来，第一反应不是感受，是找事做。这就是"忙碌麻痹"的来历——它诞生于一次"做完 = 痊愈"的错觉。

- **馆内处境（回声）**：馆里一处老漏雨，管理员放了个旧搪瓷盆接水，说「这地方老漏，习惯了」。你被叫去帮忙处理——漏雨引发的事一件接一件：先要接水（盆满了得倒）、要挪书（怕受潮得搬开）、要擦干净（地面积水得抹）。全是琐碎的具体事务。此外，这次你也翻到一本"漏雨旧事书"（别人的遗落物），借阅人栏同样涂烂。
- **实在的对比**：雨水声让你疼了——你闪回那段多重打击的日子：当年你也是这么"处理"难过的——用一件接一件的事把自己填满，以为做完就好。这一次，漏雨又给了你同样的"事"：接水、挪书、擦地。你本能地扑上去做，和当年一模一样。不是突然发酸，是同一套"用事盖情绪"的动作被同一场雨再次触发。你甚至比当年更熟练了。
- **漏水对应说明**：对得上，且是"具体事务"级别的对位——漏雨 = 你一直没补好的那道情绪裂缝（长辈离世 + 烂领导的疼，从没真正面对）；接水 / 挪书 / 擦干净 = 你忙着用"小事"不让情绪上来，把裂缝渗出来的水一遍遍擦掉，却从没想过把裂缝本身补上。你处理漏雨的方式，和处理难过的方式，是同一个动作：解决表层，绕开根源。
- **努力动作（确认）**：搬书时鼻子发酸，你几乎要停下来，跟管理员说点什么——或者干脆坐下承认「我很难过，我其实没好」。
- **习惯逃避（走捷径完成）**：你抹了把脸，把最后几本书码齐，跟他说「好了，没事」，然后去泡了杯咖啡、把灯调亮，用"把事做完＋照顾自己一下"的完成感盖过了那阵情绪。你"处理"了漏雨，也"处理"了心情——都用流程。自我域的逃避 = 用"把事做完＋一点自我安抚"压住真实情绪。（这正来自夜 B 那条根：你从"忙"里学会了把家推到以后，也学会了把情绪推到"做完再说"。）
- **正确答案（当年没做）**：当年你该允许自己停下来，承认「我很难过，我其实没好」，而不是用完成感和一点安抚把情绪咽下去。
- **管理员功能互动 + 暗示**：他递你毛巾（功能互动），没多问。后来你发现那盆被他每晚摆在老位置，盆边你那天放下的东西也在（跨夜伏笔③，物证持续存在）。
- **遗落碎片**：一阵没被承认的难过 → 遗落「感受版自己」。
- **自认播种（夜 C）**：① 借阅人墨团——漏雨旧事书借阅人栏涂烂、边缘"下次"；② 镜面映射——旧事书里"用一件接一件的事填满、以为做完就好"对应你的自我域逃避；③ 玩家是常量——你刚"做完盖情绪"，书里主人公同款动作；④ 管理员暗线——他记得你放盆边的物（伏笔③）。

## 五、第二幕·管理员之夜（夜 D/E 逼近"你就是我"，夜 F 预兆）

> 本幕不再空流程：靠"你和管理员越来越像"的寒意推进，直至温柔者反常赶人。

### 夜 D《背后楼梯》— 你就是我（闸门）

- **馆内处境**：管理员第一次带你进管理员专属区·背后楼梯 / 档案室（第二幕解锁）。说「这边有点乱，帮我理理。」——他其实在给你看他的世界。
- **故事（逼近暗喻）**：
  - 你发现他桌上也压着一本"逾期最久"的书，封皮被摩挲得发毛，和你序章那本一样旧。
  - 他讲：「我也在等一个人回来还书，等了很久，那人总说下次。」你听着，心里一种说不清的熟悉感慢慢浮上来——像在听自己的事。
  - 他做着和你一模一样的小动作：把某封信塞进抽屉说"下次处理"、把一杯茶推到"不急"的位置、被问急了就笑说"别急"。你突然意识到：管理员的每一个习惯，都是你的习惯。
- **闸门（显式机关）**：他让你帮忙理档案，你翻开那本"逾期最久的书"的借阅卡——**签名是你，借出日期正是序章续借那天**。这一刻，前面三夜埋下的种子同时回照：三本涂烂的借阅人栏、墨团边缘的"下次"是你写的；"别人"的故事全是自己的事；管理员只对"你"熟。你不是被谁宣布，而是自己"啊，原来一直是我"——拼图合拢。对应核心原则：破碎是玩家自己数出来的。
- **作用**：decoy 彻底裂向"你"，为夜 E 赶人蓄满反差。这一拍只答"你是谁 / 这些书都是你"，不抢释放戏。

### 夜 E《赶你出去的夜》— 一反常态（BT 式"因爱放你走"）

- **馆内处境**：某夜你照常来，管理员却反常地站在门口，没笑，挡着门。
- **一反常态**：他说「今晚你回去吧。别续借了。」——这是他第一次不温柔、第一次主动切断"不急"的回路。
- **双关**：「有些书，拖着不还，对谁都不好。」一语双关你的逃避，也双关他自己（他何尝不是"拖着不让你走"的那本逾期书）。
- **努力 / 逃避**：你错愕、想赖着不走，他用身体轻轻把你推出门：「走吧。」灯在身后亮着。你第一次被"赶"出图书馆。
- **作用（分离前奏）**：打破"管理员永远纵容"的预期。BT 隐喻落地——没逃的那个你，独自面对了留人的无力；而因爱你，他选择放手。为第三幕"他其实一直在用温柔留你、现在却推你走"的甜蜜反差蓄力。

### 夜 F《三物证合流》— 预兆夜

- 你终于在某次逃避后，直接问：「你到底是谁？」他不直接答，把你历夜遗落的东西摆出来——**便签、相册书、雨水盆边的物**（三个跨夜伏笔在此合流）——「这些都是你落下的，我替你收着。」
- 你仍可选逃避（把东西推回「下次整理」），但这次 Relief 几乎没了。身份揭晓的铺垫在此完成。
- **剧情同步点**：你"知道自己在逃"这件事本身，成了新的重量。循环不再是无知地转，是清醒地转。

## 六、第三幕·最后的狂欢（夜 G/H/I… 多夜，甜得刚好蓄力又有点反常）

> 玩家被赶走后习惯性地又回来了（贪恋）。此时玩家已知"是我"，却仍贪恋被稳稳等着——管理员恢复温柔，甚至更开朗，因为他知道自己要失去了，于是把最后的甜全给够。每夜快结束都留一句"明晚你还来还书吗？"，甜蜜里拽回 BE 的线。

- **节奏**：这些夜节奏快（每夜 5–8 分钟），不拖泥带水，是"最后的狂欢"。
- **管理员状态**：开朗、话多，主动带你"参观"图书馆——「这盏灯是我装的」「那排书架是我排的」「你看，这里全是给你留的」。有点炫耀，甜得有点反常（你隐约觉得他比平时更用力）。
- **收尾固定句（每夜）**：他笑问——「那明晚你还来还书吗？」既是邀请，也是把"续借 / 回来"这根 BE 的线轻轻拽回。

### 代表夜（写 2–3 个示意，结构可重复多夜）

- **夜 G《灯是谁装的》**：他带你看了那盏一直亮着的灯，说"这灯，是专门为你留的"。你心里甜，但闪过夜 E 他被你推出门的画面——甜里有刺。
- **夜 H《档案室半开的门》（给夜 Z 埋种子）**：某一夜，狂欢正甜，你"不小心"又晃到档案室半开的门前。里面那盏灯下，你序章续借的那本旧书被单独摆在显眼处，卡片上他写的一行小字：「等你还。」你没多想——此刻你明知"是我"，却仍贪恋这份被稳稳等着的甜，把这行字当作情话般收下，没去碰"该还"那一下。→ 这就是夜 Z"你带着那本书回来"的伏笔：你一直没真正还的，是序章那本。
- **夜 I《最后一盏灯前的玩笑》**：你们像老友一样把某本难还的书推到"明天"，笑闹。但他笑到最后，目光软了一下，没接话。
- **狂欢的"反常感"怎么用**：越甜，玩家越该隐隐不安——因为夜 E 已被赶过一次，这份甜是"倒计时里的甜"。每夜"明晚还来吗"都在提醒：这不会永远。

## 七、终章·夜 Z《带书回来的人》（双结局 + callback）

> 玩家选择把这一切了结，来最后一个夜晚，手里带着序章续借的那本书（夜 H 伏笔回收：你一直没真正还的，就是它）。

- **解释一切**：
  - 管理员 = 你（逾期最久的那本书 = 你遗落的版本；他是一直留下来的那个你）。
  - 整座图书馆、那盏灯、每张逾期通知 = 他留下的温柔的构造，不是欺骗，是陪伴的用心。
  - 你每一次回来、还书、陪他玩 = **最后一道逃避**：你贪恋"被稳稳等着"的感觉，不敢真的往前走（门一直没锁）。
- **两个解决（玩家选择，两结局一样权重，无主从）**：
  - **（甲）沉溺图书馆**：你把书放回书架，说「我不还了，我留下。」循环继续，甜蜜永驻。叙事诚实点明：这正是**终极逃避**——你选择永远停在温柔里，灯一直亮，你永远不长大。这是玩家可主动选的、成立的一种落点（逃避的终点），与乙并列，不以"错选"论处。
  - **（乙）告别过去（行动型 B 落点）**：你把书真正还给他、插回原位，说「该走了」。门没锁，你走出，灯留原地。各夜当年没做的（说出真话 / 回去 / 承认难过）在此刻汇成"往前走"这一个动作。你不再续借，推门离开。
- **收束句（两结局共用）**：「灯还亮着。——我可以一直陪伴在你身边，但你得自己往前走了。」
- **BE 温度（不可滑偏）**：两个版本的你都竭尽全力，只能到这里。限度温柔，非致郁；不是"你痊愈了"，是"你终于肯往前"。甲、乙都是"只能到这里"的诚实落点，只是方向不同。

### 终章 callback（第一幕 A/B/C 正确答案闪回 · 知≠行）

> **设计意图**：终章要把第一幕夜 A/B/C 的"正确做法 / 成长部分"做闪回 callback，说明主角其实一直知道要怎么做——但情绪和压力不会因为他懂了道理就凭空消失。这不是救赎、不是"你痊愈了"，恰恰相反：它把"只能到这里"的限度从"不懂"改成"懂了也还是难"，反而更深、更诚实地落住了 BE 温度。
> **关键纪律**：callback 必须落在**知 ≠ 行**的缝隙，绝不能写成"现在三件都做到了"。闪回要同时呈现两样——① 努力动作（当年你摸到笔 / 悬在拨通键 / 鼻子发酸想停）证明你**知道且想做对**；② 情绪与压力如何压过道理（怕 / 无话可说 / 做完焊成本能）。落点 = 「我从来都知道，只是懂了也还是难」。

- **放置位置**：在 ① 揭晓"管理员 = 自己"之后、③ 抉择分离之前，作为通向"往前走"的「知」段落；可与 ② 逃避开心的夜叠层（甜蜜里浮出三封迟到的答案）。用夜 F 已合流的三物证（便签 / 相册书 / 雨水盆边物）作触发物。
- **三封迟到的正确答案（闪回内容，复用各夜已锁"正确答案"）**：
  - **便签 · 夜 A（关系域）**：闪回你摸到笔、写了两句真话又揉掉、送信箱了事。旁白——「我那时就知道了，那句话该说给该说的人。可电话拿起来，情绪先一步堵住嘴。知道该说，和说出口，中间隔着一整场怕。」
  - **相册书空位 · 夜 B（家庭域）**：闪回你手指停在年夜饭空位、号码悬在拨通键、只发了红包。旁白——「我那时也知道该回去、该说『我想你们 / 那年我没回来我后悔』。可『忙』背后那股压力——怕无话可说、怕被旧地图接不住——比道理重。知道该回，和回得去，是两件事。」（此处只做「知」的承认，夜 B 专属的"理解父母 → 放下傲慢 → 和解"留后续，不在此完成。）
  - **雨水盆边物 · 夜 C（自我域）**：闪回你鼻子发酸想停下、却抹了脸说「好了没事」去泡咖啡。旁白——「我甚至知道该承认『我很难过、我没好』。可那阵压力把『做完』焊成了本能，道理在它面前轻得像灰。知道该停，和停得下来，隔着我自己。」
- **统一落点（本段核心句）**：这三段不是「我终于做到了」，是「我终于承认——**我从来都知道要怎么做，只是情绪和压力不会因为我懂了道理就消失**」。所以那些版本（表达版 / 归家版 / 感受版自己）遗落了，不是因为我蠢、不懂，是因为懂了也还是难。这正是「只能到这里」的诚实：限度不在无知，而在生而为人那点沉甸甸、绕不开的难。
- **与乙 / 甲结局衔接**：
  - **（乙）告别**：「门没锁你走出」不是「我痊愈了」，是「我带着这份『知道却曾做不到』的自己，肯往前走了」。A/B/C 当年没做的，有的已经迟了（那封信、那年爸生病），但「往前走」这一个动作，是此刻唯一还来得及的「正确答案」。
  - **（甲）沉溺**：则把三封答案连同灯一起留下，永远停在「知道却贪恋被等」的温柔里——也是「只能到这里」的一种，只是方向不同。

## 八、跨夜回照机制（修复"外围 / 单元剧"问题）

- **馆内物证持续存在**：便签（服务台抽屉）、相册书（"待你再来看"架位）、雨水盆（每晚老位置）——都是图书馆空间状态，不是别人的故事。
- **通知单累积**：每张逾期单带手改痕迹（管理员手写划掉又写），后期玩家发现笔迹越来越像自己。
- **管理员记忆碎片**：他偶发掉落"他也是被遗忘的自己"的碎片，跨夜收集 → 终章拼图。
- **氛围随内心变**：灯色、雨声、台词密度随觉悟微调（夜 E 赶人时灯骤暗；第三幕狂欢时灯偏暖偏亮，但暖得反常）。

## 九、自认机制（播种 → 回收）

> 自认机制是两条回收的总闸：① 补全角色最终性格走向（从"无意识逃"转轴到"清醒地转"）；② 让第一幕所有铺垫（墨团 / 镜面 / 玩家常量 / 管理员暗线）在 D/F/Z 一次性回照合拢，故事铺垫回收完整。分寸：不"明显提示"——A/B/C 绝不点破"其实是你"；但要让玩家自己拼出"是我"，靠可回照的物证种子，而非 D 幕空降反转。自认是情绪上的诡异熟悉（uncanny）：慢慢觉得不对劲、最后被戳穿，不是逻辑推理出"是我"。它由玩家在夜 D 闸门自己合拢。

**写作纪律**：
- ② 镜面"够像别人、够不像自己"——细节可重合，但情绪 payload 不提前给（夜 A 只闪回"我当初就是这么逃的"一沉，不展开；夜 B/C 同理只到"一沉"为止）。
- ③ 的呼应要轻到玩家可忽略（伞 / 红包 / 做完都是随手动作），重了就泄底。
- ④ 前期必须读成职业习惯，不能让管理员流露出"他在盯你"。

### 9.2 第二层 · 夜 D 闸门（显式机关，靠种子命中注定）

- **机关**：档案室那本"逾期最久的书"借阅卡——签名是你，借出日 = 序章续借那天。
- **但它不能孤立出现**：玩家在这一刻被前面的种子同时回照击中——① 三本涂烂的借阅人栏、墨团边缘的"下次"是自己写的；② "别人"的故事全是自己的事；③ 管理员只对"你"熟。
- **合拢方式**：不是管理员宣布"你就是借阅人"，而是玩家自己"啊，原来一直是我"——拼图合拢，不是被告知。对应核心原则：破碎是玩家自己数出来的。
- **闸门纪律**：这一拍只答"你是谁 / 这些书都是你"，不抢释放戏；夜 E 赶人、终章 callback 才分别接"放你走"与"知 ≠ 行"。

### 9.3 第三层 · 角色弧转轴 + 铺垫回收总闸

### 9.4 给写作者的密度检查（防过埋 / 防欠埋）

- 每夜**至少埋 ① 借阅人墨团 + ④ 管理员暗线**两条常量（最便宜、最隐蔽，几乎零成本）；
- ② 镜面映射每夜一条（A 关系 / B 家庭 / C 自我，已锁域分配，不增不减）；
- ③ 玩家常量呼应每夜一条（可极轻，玩家忽略也不影响）；
- **红线**：任何一夜间接出现"这好像是我""你就是本人"之类的点破句 = 违规，打回重写。自认只能发生在夜 D 闸门，且由玩家自己合拢。
- **标志物候选（不锁定）**：借阅人墨团里的手写"下次"可提炼为贯穿标志物（与逾期通知卡同源），待夜 A–C 文字自然浮现后再定。
