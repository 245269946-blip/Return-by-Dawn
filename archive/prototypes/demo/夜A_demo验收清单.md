## 你点着看时，按这条线索走

**① 开场** → 读通知 / 或先揉一团再展平，都能进馆。

**② 四片区域随便逛**（借阅台 / 阅览区 / 书库深处 / 雨夜门廊）
- 每片区域里有**可点的物件**（一张逾期单、一盏旧灯、夹在书里的信、门廊的伞……）。
- 点物件会冒出一句话 + 右下角管理员偶尔搭一句（他话很少，这是刻意的）。
- 旧灯可以追问「谁？」；便签可以**写真话 / 写安全的话 / 什么都不写**（三选一，且选过一次就锁死，再点只复述、不叠加）。

**③ 收束门槛（关键）**
- 必须同时拿到**「信」+「写着你名字的通知单」**两件关键线索，区域底部才会出现「走到灯下」按钮。
- 只拿到其中一件，按钮是灰的、点不动——这关是故意卡的。

**④ 结局三选一**：归还 / 带走 / 销毁，三种不同收尾。选完就结束，不再有操作。

**⑤ 存档验证（B）**
- 逛到一半，直接关掉页面；再打开 → 出现「继续上次」。
- 点继续：你之前逛过的区域、捡到的线索、选过的便签，全都在。
