# 夜 A 叙事地基梳理（地基版 · 待碰撞 · 非最终框架）

> 本文件只做一件事：把「角色底色 → 人物故事 → 母亲故事线」的逻辑地基钉清楚，并划清夜 A 只取哪一层。不直接落地 JSON、不做完整关卡设计——按你的要求，先别跨太大步。

## 0. 先认错（方法层）
上一版 v2 把「用户举的结构示例」（妈妈登记本、书库迷宫）直接当成夜 A 的剧情写进去，导致夜 A 成了示例拼凑、不伦不类。根因：把「全局结构建议」和「单关内容」混为一谈。本文件先把两层剥开。

## 1. 已锁定的全局事实（通用层，不再动）
- **脊柱**：图书管理员 = 逾期最久的那本书 = 玩家自己遗落的版本（后期夜 C / 终章揭晓）。
- **主角底色**：核心已碎、默哀心死；表面寻书/和解，底层是「自己遗落了自己」。
- **BE 温度**：真实、温馨、所有人竭尽全力做到无憾、但只能到这里（限度的温柔，非致郁）。
- **第一幕** = 纯 decoy 沉浸期，图书馆内，轻、不承重。
- **结构层（已确认通用）**：柔性触发（无固定揭晓点）／四区固定（借阅台·阅览区·书库深处·归还箱）／锈湖式节点推进／回忆=探索本身／每区 2–4 互动点且同点跨夜异用。
- **母亲章** = 投射式小结局装置的旗舰范例，明确「**非脊柱**」。

## 2. 角色底色（锁定，无须再议）
玩家不是「来还书的外人」，而是**把自我遗落在图书馆里的人**。图书馆是角色自己想象虚构的空间，是留给破碎自己的「温柔的构造」。表面在帮人找书、在和解，底层那个核心早就碎了、默哀心死。夜 A 呈现的应该是「一个还在找、但不知道找的是自己」的状态。

## 3. 人物故事（真正的空缺，必须先与你对齐）
目前只有「底色」，没有「具体是谁、发生过什么」。这是夜 A 之前必须钉死的地基——没有它，母亲线、信、伏笔都没有重力。候选方向（**供碰撞，非定稿**）：
- 玩家曾是一个「把什么都照看得很好」的人（对照现在的碎）；母亲是早年最重要的牵系。
- 某次断裂（不必在夜 A 说破是什么）让玩家把自己遗落在了这座图书馆里。
- 夜 A 不揭断裂，只呈现「还在找」的状态。

→ **需要你拍板**：断裂的性质（丧失 / 自我放弃 / 关系冻结？）、玩家与母亲关系的基调（温暖但够不着 / 疏离 / 单方面依赖？）。这两点不先定，后面全是浮沙。

## 4. 母亲故事线（节点性，非脊柱，且不全在夜 A）
纠正一个倾向：母亲线若整段塞进夜 A，会同时击穿两条已定事实——「第一幕轻、不承重」和「母亲 = 非脊柱旗舰装置」。正确位置是**跨夜节点结构**，夜 A 只承载「第 0 个种子节点」：
- **节点 0（夜 A 种子）**：借阅登记本上，妈妈很多年前借过**同一本**书，页边有一行很轻的批注。玩家看到时只是「诶，巧合？」，无情绪 payload。
- **节点 1（中后段某夜）**：关于母亲的温暖记忆结晶，你以为是重新接上了。
- **节点 2（后段）**：投射裂开——那从来不是她来够你，是你伸向她却够不到；温暖是真的，但只能到这里。
→ 这就是「投射式小结局」装置的标准落点，落在更重的夜，不在夜 A。

## 5. 夜 A 只取哪一层（我的提案，待碰撞）
- **保留原标题《夹在书里的信》**——信 = 自我碎片，贴合脊柱，且天然是 decoy（你以为信是陌生人写的）。
- **中心谜（decoy）**：一本书里夹着信，你以为是某个陌生逾期者的故事；探索后拼出一个「别人」的轮廓，轻量伪 HE 收束。信其实是你过去版本写给自己，但夜 A **不揭**。
- **母亲线**：仅「节点 0 种子」一个互动点（登记本那一行批注），轻、不承重。
- **书库迷宫示例**：建议**移出夜 A**（世界铺陈太重；且「不同人的书 = 都是你」是后期回收装置，不是第一夜该扛的）。夜 A 的书库深处只做轻量探索 + 一个伏笔物。
- **探索密度**：四区各 2–4 互动点、同点跨夜异用——这是下一步内容设计的原则，等 §3 钉死后才铺，不和地基混做。
